## 景别（人物在画面里占多大）

| 专业词 | 人话解释 | 什么场景用 |
| --- | --- | --- |
| 大远景 / 全景 | 整个场景都能看到，人物很小 | 交代环境、表达孤独、震撼感 |
| 远景 | 人物全身入画，周围有空间 | 看清动作和环境的关系 |
| 中景 | 腰部往上，能看清手势和表情 | 最日常、最安全的景别 |
| 中近景 | 胸部往上 | 强调上半身动作和表情 |
| 近景 | 肩膀往上，脸部为主 | 突出表情、对话感 |
| 特写 | 脸填满画面 | 强烈情绪、细节 |
| 大特写 | 只有眼睛、嘴、手等局部 | 极致情绪、戏剧张力 |

### 景别的"立人设"用法（不靠正脸传达身份）

| 手法 | 怎么写 | 适合什么 |
| --- | --- | --- |
| 不拍正脸 | 全程只给侧脸、背影、剪影 | 神秘感、未揭面的关键人物 |
| 局部特写串联 | 大特写只拍配饰（袖扣/领带/手表/徽章/手势） | 不靠对白立"有身份/讲究"的人设 |
| 背影跟拍 | 中近景跟在人物背后，看他与环境互动 | 让观众跟着人物进入场景，又保留神秘感 |

## 机位角度（相机从哪个角度拍）

| 专业词 | 人话解释 | 心理含义 |
| --- | --- | --- |
| 平视 | 相机和主体眼睛一样高，最中性 | 平等、客观 |
| 俯拍 / 俯视 | 相机从上往下看 | 主体**弱势**、被审视、孤立、渺小 |
| 仰拍 / 仰视 | 相机从下往上看 | 主体**强势**、压迫感、英雄气、眩晕 |
| 鸟瞰 | 完全从正上方往下 | 上帝视角、战术展开 |
| 荷兰角 | 画面是斜的（相机绕镜头中线偏转） | 不安、失衡、眩晕、精神扭曲 |

**用机位高低表达权力关系**：拍强势方用平视/微仰视、拍弱势方用俯视。同一场对话两人都用平视会丢失权力差。

### 心理角度（观众身份）

几何角度（俯/仰/平/荷兰/鸟瞰）讲"相机在哪"。心理角度讲"观众通过这个镜头是谁"——旁观者、还是角色本人。

| 类型 | 怎么拍 | 观众体验 | 何时用 |
| --- | --- | --- | --- |
| **客观镜头** | 旁观式拍摄、不模拟任何角色视点 | 公平、理性、冷静 | 大部分叙事场景的默认。观众以观察者身份介入 |
| **主观镜头**（POV） | 模拟画面主体的眼睛和视线 | 沉浸、代入、生理感受 | 角色被打击/眩晕/喝醉/被偷看时让观众"成为他"。或反向：让观众"成为偷看的人" |

写主观镜头 prompt 时明确写："第一人称视角、模拟人物 X 的视点、画面有轻微呼吸感起伏"。

## 构图（画面怎么安排）

| 专业词 | 人话解释 |
| --- | --- |
| 居中构图 | 主体放正中间，稳重、仪式感、对称 |
| 三分法 | 主体放在画面的 1/3 线上，更自然、有呼吸感 |
| 对称构图 | 左右或上下对称，秩序感强 |
| 留白 | 画面某一侧空出来，制造情绪、给视线方向让位 |
| 前中后景 | 画面有三层空间——前景挡一点、中景是主体、后景是环境 |
| 浅景深 / 背景虚化 | 主体清晰，背景糊掉，突出主体 |
| 深景深 | 前后都清晰，信息量大，纪实感 |
| 框架构图 | 用前景的门、窗、树枝框住主体，增加层次 |

### 视线留白方向（同样的人放同样的位置，左留白和右留白完全是两种情绪）

| 用法 | 怎么放 | 效果 |
| --- | --- | --- |
| 顺视线留白（默认） | 人看左边 → 人放在右边 1/3，左边大片留白 | 自然、舒服、有呼吸感 |
| 反向视线留白（压抑感） | 人看左边 → 人也放在左边 1/3，前方撞墙 | "面壁感"、压迫、不安、内心封闭 |

### 立体感四要素（避免画面变扁平贴纸）

写完一张图觉得"很扁、像 AI"通常是这四件事都没做：

1. **透视**：用焦段建立"近大远小"（24mm 强透视、50mm 自然、70mm 压缩）
2. **虚实**：用浅景深拉开主体与背景（参考下面"景深控制"一节）
3. **明暗**：用侧光/逆光制造亮部和暗部的层次，避免大顺光
4. **遮挡**：前景放点东西挡一下（虚化的树叶、栏杆、半透明屏风、玻璃反光）

王家卫式电影感构图 = 前景遮挡 + 镜面反射 + 高反差冷暖光 + 浅景深，四件事一起上。

### 景深控制（怎么让背景糊起来）

影响景深浅深的**三个独立因素**，写 prompt 时挑一个最具体的写：

| 因素 | 让景深更浅（更虚化）的方向 |
| --- | --- |
| 光圈 | 大光圈（f/1.2、f/1.8、f/2.0） |
| 焦段 | 长焦（70mm、85mm、135mm） |
| 距离 | 相机贴近主体 |

**移焦（rack focus）**：同一画面里焦点从前景人物滑到后景人物身上，暗示注意力转移或人物关系变化。
**裂焦**：通过特殊镜头让画面同时存在两个清晰焦平面（前景和后景同时清楚），有强烈的形式感。

## 光线（画面的光从哪来、什么颜色、硬还是柔）

### 光的方向 / 软硬 / 色温（基础）

| 专业词 | 人话解释 |
| --- | --- |
| 顺光 | 光从主体正前方来，脸全亮没阴影，平、没立体感 |
| 侧光 | 光从侧面来，半张脸亮半张脸暗，立体感强 |
| 逆光 | 光从主体背后来，主体变剪影或边缘发光（轮廓光） |
| 顶光 | 光从正上方，眼窝会有阴影，硬、戏剧感 |
| 低色温 / 暖光 | 画面偏暖黄，像夕阳、钨丝灯、烛光、橘色路灯（约 3200K） |
| 高色温 / 冷光 | 画面偏冷蓝，像阴天、月光、荧光灯、雪地（约 5600K） |
| 硬光 | 光源小而强，阴影边缘锐利，戏剧性 |
| 柔光 | 光源大而散，阴影边缘柔和，温柔感 |
| 黄金时刻 | 日出后 / 日落前半小时，光温暖柔和、色彩饱和 |
| 蓝色时刻 | 日落后 / 日出前那段天空呈深蓝色的时间 |
| 体积光 / 丁达尔光 | 能看见光束穿过空气（灰尘、雾、水汽中） |
| 霓虹光 | 彩色霓虹灯的反射，通常红蓝对比，赛博朋克感 |

### 硬光 vs 柔光的时间叙事

硬光（直射光）和柔光（散射光）能传递不同的时间信息和情绪基调：

| 用法 | 视觉特征 | 时间暗示 | 情绪 |
| --- | --- | --- | --- |
| **硬光主导** | 阴影边缘锐利、明暗反差大、影子短 | 正午时分 | 戏剧化、压迫、明确 |
| **柔光主导** | 阴影边缘柔和、明暗过渡丰富、影子长 | 清晨或黄昏 | 温暖、抒情、回忆 |
| **从硬光转柔光**（时间感设计） | 同一场景前段硬光、后段柔光 | 一段时间过去（从正午到傍晚）| 情绪从外露到内敛 |

经典案例：《蝙蝠侠：侠影之谜》用柔光拍童年闪回（双亲被杀前）、那一段感觉美好；之后所有镜头都换成硬光、暗示创伤后的人物状态变化。

写 prompt 时不要只写"光"、要明确："正午硬光、影子短而锐利"或"黄昏柔光、影子长而柔和"。

### 9 种主光打法（一盏灯放在不同位置带来的情绪）

写 prompt 时不要只写"侧光"——把面部阴影的形状直接描述出来，模型才出得准。

| 主光名 | 灯的位置 | 面部阴影特征（写进 prompt 的关键） | 适合的情绪 / 角色 |
| --- | --- | --- | --- |
| 纯逆光 | 主体正后方 | 面部完全在阴影里，只剩头发和肩膀的轮廓发光 | 神秘、反派、悬疑 |
| 3/4 背光（侧逆光） | 侧后方 45° | 脸大部分在暗部，只勾出脸颊轮廓和下颌线 | 硬汉、宿命、酷感 |
| 纯侧光（阴阳脸） | 正侧方 90° | 半张脸全亮、半张脸全暗，分界线在鼻梁中线 | 纠结、两面性、显瘦 |
| 伦勃朗光 | 侧前方 45° 略高 | 暗侧脸颊上由鼻翼和颧骨阴影围出**倒三角形光斑** | 经典油画感、王家卫常用 |
| 大顺光（平光） | 正前方 | 面部均匀照亮，无任何阴影 | 平、证件照、综艺、影视剧少用 |
| 教父光（顶光） | 正上方略偏前 | 眼窝、鼻下方、下巴下浓重阴影 | 威严、压迫、上位者 |
| 底光（恐怖光） | 正下方往上打 | 阴影方向反转——鼻下变亮、额头变暗 | 诡异、惊悚 |
| 蝴蝶光（派拉蒙光） | 正前方偏上往下 | 鼻子正下方对称蝴蝶形阴影 | 显瘦、唯美、好莱坞女星 |
| 动态走光 | 灯在面部前方移动 | 光影在脸上快速流转 | 紧张、梦幻、情绪失控 |

### 进阶光线概念

| 专业词 | 人话解释 | 怎么写进 prompt |
| --- | --- | --- |
| 逻辑光源（美术光） | 画面里**肉眼可见**的发光道具：台灯、窗户、蜡烛、霓虹招牌、打火机 | 写"右上角窗户透进的冷白光" 比写"侧光" 模型更可控 |
| 黑柔滤镜 | 高光柔化、暗部依然扎实，去 AI 数码锐利感，胶片高级灰 | "高光柔化扩散、暗部保持扎实、轻微胶片光晕" |
| 白柔滤镜 | 整体朦胧、强烈梦幻感 | "整体朦胧光晕、梦幻柔焦" |
| 蜂窝聚光 | 灯光只照亮局部、不外溢 | "光只打在脸部、背景保持暗部" |

## 氛围 / 情绪基调

| 专业词 | 人话解释 |
| --- | --- |
| 电影感 | 宽画幅 + 浅景深 + 有戏的光 + 色调统一 |
| 日常感 / 纪实感 | 平视、自然光、没有刻意构图、稍微随手 |
| 赛博朋克 | 霓虹灯、冷暖对比强、雨后反光、未来感 |
| 胶片感 | 颗粒、轻微色偏（常偏绿或偏青）、高光柔和不爆 |
| 极简 | 元素少、大留白、颜色少 |
| 戏剧性 | 光影对比强、角度奇特、情绪浓烈 |
| 梦境感 | 柔光、轻微过曝、色彩偏粉紫、边缘发光 |
| 纪录片感 | 平视、中景、自然光、构图不过于讲究 |

## 颜色 / 色调

| 专业词 | 人话解释 |
| --- | --- |
| 饱和度高 | 颜色鲜艳浓烈 |
| 饱和度低 | 颜色灰淡、克制 |
| 对比度高 | 黑的更黑、白的更白、反差大 |
| 对比度低 | 灰蒙蒙的、柔和过渡 |
| 冷暖对比 | 画面同时有冷色（蓝青）和暖色（橙黄），戏剧感 |
| 单色调 | 整个画面偏一种颜色（青、橙、品红等），风格化 |
| 莫兰迪色 | 低饱和度、灰粉灰绿灰蓝为主，文艺 |
| 高饱和撞色 | 红蓝、黄紫等高饱和对撞，视觉冲击 |

## 质感 / 材质（物体表面的感觉）

| 专业词 | 人话解释 |
| --- | --- |
| 丝绒 | 绒面质感，光打上去有柔和反光 |
| 哑光 | 不反光、雾面 |
| 金属光泽 | 反光强、有高光条 |
| 磨砂 | 粗糙雾面、不反光 |
| 湿润 | 表面有反光、像刚洒过水 |
| 皮革 | 有光泽但不如金属强、有纹理 |
| 棉质 | 柔软、无反光、有褶皱 |

## 运镜（主要视频用，静图里只用来"模拟镜头感"）

| 专业词 | 人话解释 |
| --- | --- |
| 推镜 | 相机向主体靠近 |
| 拉镜 | 相机远离主体 |
| 摇镜 | 相机原地转动（左右 / 上下） |
| 跟镜 | 相机跟着主体移动 |
| 升降 | 相机上下垂直移动 |
| 环绕 | 相机围绕主体转圈 |

**注意**：静态图提示词里一般不写运镜动作，但可以用"模拟从 XX 角度拍摄的瞬间"这种描述，让画面有镜头感。

## 常见陷阱词（模型听不懂，不要用）

以下这些词模糊、空泛，模型无法精确执行，写了等于没写：

- ❌ 大师级、大师作品
- ❌ 电影级、电影质感（替代：浅景深、胶片颗粒、冷暖对比）
- ❌ 史诗感、震撼、宏大
- ❌ 超高清、8K、4K、高清（模型不会因为你写 8K 就出更清楚的图）
- ❌ 精致、完美、极致
- ❌ 获奖摄影作品、顶级摄影

**替代思路**：把空泛词拆解成"具体可见的画面特征"。比如"电影感"→"浅景深 + 冷暖对冲光 + 胶片颗粒 + 统一色调"。
