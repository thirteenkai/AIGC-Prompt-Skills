## 使用纪律

1. **不要在【最终提示词】里直接写"王家卫风格"**。模型对导演名字理解模糊。要把范式拆成"具体可见的画面要素"再写。
2. **不混搭**。一张图里同时写"王家卫的冷暖对冲 + Wes Anderson 的对称构图 + 是枝裕和的低机位"，模型迷茫。挑一种范式做主、其他作微调。
3. **范式不是全套搬运**。挑 3-5 个最关键的要素、不要 8 个都用。
4. 用户没说要某种范式时、不要主动塞导演风格。本文件是被动查询用、不是默认应用。

## 范式 1：王家卫

### 关键要素

- 浅景深 + 长焦压缩空间
- 冷暖光对冲（霓虹蓝/红 + 暖黄路灯/钨丝灯）
- 雨水反光 / 玻璃反光 / 镜面反射
- 前景遮挡（虚化的窗框、霓虹招牌、雨幕、窗帘）
- 人物偏画面一侧 + 大面积留白指向"画外"
- 胶片颗粒 + 黑柔滤镜（高光柔化、暗部扎实）
- 偏色（青绿调或品红调，不是中性）
- 抽帧 / 拖影感（视频时用，图片用"动态模糊"模拟）

### 适用题材

夜戏、暧昧、压抑、都市孤独、错过、回忆。

### prompt 翻译模板（图片）

```
[人物 + 动作]，[场景细节]。
长焦压缩空间，浅景深，前景[具体遮挡物]虚化在画面[位置]，
主光为[暖黄/冷蓝]逻辑光源（[具体来源如窗户/霓虹/路灯]），
背景有[反差色]环境光。
胶片颗粒，黑柔滤镜效果（高光柔化扩散、暗部扎实），
画面整体偏[青绿/品红]调，低饱和度。
人物位于画面[左/右] 1/3 处，对侧大面积留白。
```

### 反例

❌ "王家卫风格、电影感、史诗级"——三个空泛词、模型不会执行。

## 范式 2：是枝裕和

### 关键要素

- 低机位（相机略低于人物视线、模拟"坐着看"的视角）
- 自然光主导（没有强烈灯具痕迹）
- 中景为主，少特写
- 浅景深，但不夸张（背景虚化适度、人物清晰）
- 水平构图、画面平静
- 暖色调但克制（不是鲜艳暖、是"日常的暖"）
- 室内场景多用窗外光透入
- 人物间留呼吸距离

### 适用题材

家庭、日常、温情、儿童视角、缓慢的生活观察。

### prompt 翻译模板

```
[人物动作和场景]。
低机位（相机略低于人物视线），中景，水平构图。
自然光为主，[具体光源如"窗外透入的午后阳光"]，
柔和暖色调但低饱和、不张扬，
背景适度浅景深，主体清晰、环境信息保留。
胶片质感、整体平静无戏剧光影。
```

### 适合的具体画面

- 餐桌上一家人围坐
- 孩子在榻榻米上玩玩具的瞬间
- 老人在窗边发呆
- 兄妹一起做饭

## 范式 3：Wes Anderson

### 关键要素

- 严格的对称构图（主体在正中央、背景元素左右对称）
- 平视机位（相机和主体水平）
- 高饱和度但限制色板（一张图通常 2-3 个主色）
- 鲜明的色块对比（粉 + 绿、黄 + 蓝、米色 + 红）
- 扁平化构图（前后景压缩在一个平面上）
- 极简或装饰性强的背景（不是杂乱的真实场景）
- 人物正面 / 完美侧面（不是四分之三侧）
- 道具摆放整齐（强迫症式精确）

### 适用题材

童话感、复古、强烈视觉风格、装饰性强的画面。

### prompt 翻译模板

```
[人物或主体]位于画面正中央，严格对称构图，
背景[元素1]在左、[元素1镜像]在右、上下也对称。
平视机位，正面[或正侧面]视角。
色板限定 [主色1] + [主色2] + [可选主色3]，高饱和度，
色块边界清晰，画面扁平化（前后景压缩），
背景[简洁/装饰性强的具体场景]，所有道具摆放整齐对称。
```

### 慎用提示

写实题材慎用。Wes Anderson 美学是"反真实"的、强行套到写实剧只会显得做作。

## 范式 4：罗伊·安德森（北欧冷喜）

### 关键要素

- 极远景 + 全广角（人物在远处、空间巨大）
- 冷调灰白（去饱和、像褪色明信片）
- 单点光源 / 平铺漫射光（无戏剧光）
- 人物面无表情、姿态僵直
- 极简空间（一片大墙、一个长走廊、一个空房间）
- 用空间的"巨大"反衬人物的渺小
- 画面停滞感（像一张明信片、不像一个瞬间）

### 适用题材

存在主义、孤独、荒诞冷喜剧、人在时代里的渺小。

### prompt 翻译模板

```
极远景、人物在画面[远处某位置]、占画面 1/8 不到。
背景是[巨大的单色空间，如灰白色墙面/空旷走廊/无人厅堂]。
冷调灰白色调，整体去饱和（像褪色照片），
单点漫射光、无戏剧明暗，
人物面无表情、姿态僵直，
画面有强烈的停滞感、像一张明信片。
```

## 范式 5：诺兰

### 关键要素

- IMAX 感（开阔水平 + 纵深空间）
- 冷蓝调（IMAX 标志性的钢青蓝）
- 高对比度、暗部扎实
- 大幅运动 / 物理冲击的画面定格
- 时间错乱构图（沙漏、镜面反射、时钟元素）
- 写实质感（不夸张、不卡通）
- 没有暖色（即使日光戏也偏冷）

### 适用题材

科幻、悬疑、宏大叙事、时间主题。

### prompt 翻译模板

```
[场景和动作]。
广角镜头、开阔水平视野、纵深空间感，
冷蓝调（钢青色为主、几乎无暖色），
高对比度、暗部扎实不漂浮，
写实质感（无卡通化），
[如适用："画面中包含[镜面反射/沙漏/时钟]元素，暗示时间主题"]。
```

## 范式 6：贾樟柯 / 侯孝贤（华语写实）

### 关键要素

- 长焦中远景（人物在环境里、不放大）
- 自然光、阴天感、低饱和
- 中国/东亚特有的城乡环境（小镇、城中村、县城、菜市场、铁路边）
- 人物动作日常（吃饭、走路、抽烟、闲聊）
- 画面有"未经修饰"的真实粗粝感
- 不用浅景深（前后都清晰）
- 中焦或长焦但不压缩到 Wes Anderson 那种扁平

### 适用题材

中国/东亚写实、社会观察、文学性叙事。

### prompt 翻译模板

```
[人物 + 日常动作 + 中国小镇/县城/城中村具体场景]。
中远景（人物在环境里、占画面 1/4 到 1/3）、深景深（前后都清晰），
长焦但不压缩，自然光、阴天漫射感，
低饱和度、轻微偏黄绿调（南方）或偏灰冷调（北方）。
画面有未修饰的粗粝感、街道/墙面/电线/招牌细节都保留。
```

## 范式 7：David Fincher

### 关键要素

- 极低饱和（几乎黑白但有微弱青绿调）
- 暗部极暗、亮部不爆
- 精确构图（每个元素都在该在的位置、像广告片）
- 阴影硬朗（光源方向明确）
- 室内多用顶光 + 单点光源
- 城市夜景 + 雨 + 反光
- 人脸常常半在阴影里

### 适用题材

悬疑、犯罪、心理惊悚、广告片质感。

### prompt 翻译模板

```
[人物动作和场景]。
极低饱和度（几乎单色但偏冷绿调），
暗部极暗扎实、亮部克制不爆，
单点硬光源（[具体如台灯/顶灯/窗户]），人脸半在阴影里。
精确构图、每个画面元素位置感强。
画面有广告片式的精致质感（不是日常感）。
```

## 范式 8：奉俊昊

### 关键要素

- 信息密度高（一个画面里有多层叙事）
- 上下/前后空间错位（同一画面里富人在上、穷人在下；或前景一件事、后景另一件事）
- 冷暖混合（不是对冲、是混杂）
- 写实但有一丝戏剧化（光稍微比真实强一档）
- 道具/环境细节大量保留（用环境讲故事）

### 适用题材

社会议题、阶级、悬疑、多重叙事。

### prompt 翻译模板

```
[主场景 + 主人物动作]。同一画面里 [前景/后景或上下] 还有 [次要元素或人物动作]、暗示 [对比关系如阶级/年龄/生死]。
冷暖混合光（[来源1]提供暖、[来源2]提供冷），
写实质感、但光线稍强于自然光（戏剧化一档）、
环境细节大量保留，构图层次明显。
```

## 怎么和用户对话（识别她想要的范式）

用户说"电影感"时、不要直接套王家卫。先问：

- 你想要的是冷感（蓝灰）还是暖感（橘黄）？
- 镜头里有人在动还是静止？
- 是城市还是乡村 / 是夜还是日？
- 想要克制平静还是戏剧浓烈？

根据回答匹配：

| 用户描述 | 推荐范式 |
| --- | --- |
| 暖、平静、家庭、日常 | 是枝裕和 |
| 冷、暧昧、夜戏、孤独 | 王家卫 |
| 强烈视觉风格、对称、复古童话 | Wes Anderson |
| 冷、宏大、悬疑、科幻 | 诺兰 |
| 中国小镇 / 写实 / 文学感 | 贾樟柯/侯孝贤 |
| 犯罪 / 悬疑 / 广告片精致 | Fincher |
| 社会议题 / 阶级 / 多线叙事 | 奉俊昊 |
| 北欧冷感 / 渺小 / 荒诞 | 罗伊·安德森 |

## 范式 9：乌尔善（神话宏大）

### 关键要素

- 神话/古风奇幻世界的具象化（宗门、神兽、法术、巨型场面）
- 高对比度色彩（金红 / 玄黑 / 暗紫）
- 仪式感构图（中轴对称、人物居中）
- 镜头偏重"震撼一击"的瞬间定格
- 真实摄影质感（不是卡通、是"神话被实拍"的感觉）

### 适用题材

封神类、东方神话、宏大法术对决、宗门祭典。

### prompt 翻译模板

```
[人物 + 神话动作或仪式]、[巨型场景如祭坛/神兽/法阵]、中轴对称构图、人物居中。
高对比度暗金/玄黑/暗红色调、戏剧化硬光（仪式感强）、真实摄影质感（非卡通）、
画面中包含 [神话元素：神兽轮廓 / 浮空法器 / 流光符文]、定格在动作高点。
```

适合：剧组里的"高燃定场镜头"、神话感强烈的瞬间。

## 范式 10：张艺谋（色彩形式感）

### 关键要素

- 单一主色覆盖大面积画面（红 / 青 / 黄）
- 强烈对称构图、人物在画面正中
- 水墨意境（特别是《影》系列）：黑白灰 + 留白 + 水
- 仪式感动作（武术、祭礼）
- 远景或大全景多于近景

### 适用题材

仪式感强的瞬间、水墨意境、色彩奇观、宫廷祭典。

### prompt 翻译模板

```
[主体动作 + 单一主色场景]。[主色名（如赤红/青灰/雪白）]覆盖画面大面积、强烈对称构图、人物在画面正中。
[如果是水墨风："黑白灰为主、大量留白、画面有水或雾的元素"]。
远景或大全景、镜头静止或缓慢推近、形式感大于真实感。
```

## 范式 11：胡金铨 / 徐克 / 邵氏港台（古早武侠）

### 关键要素

- 胶片颗粒感、轻微偏黄/偏暖的色调
- 快速变焦放大（邵氏经典手法、1 秒内完成、表现冲击感）
- 飘逸的衣袍 + 慢动作打斗
- 留白构图（人物常在画面 1/3 处、留出大量天地）
- 自然光为主、不刻意打灯
- 写意而非写实

### 适用题材

武侠、古早港台风格、江湖意境。

### prompt 翻译模板

```
[人物 + 武侠动作 + 古风场景]。胶片颗粒、轻微偏黄暖色调、自然光主导、不刻意打灯。
人物位于画面 [左/右] 1/3、对侧留出大面积天地空间、衣袍飘逸。
[如果是动作戏："快速变焦放大（1 秒内）+ 慢动作打斗"]。
写意感强、不追求写实细节。
```

## 找对标策略（没想法时的可执行方法）

用户说"我想要 X 那种感觉"但你或她都说不清"X 是什么"时：

1. **找同题材现有影视剧**：剧本是宅斗 → 看《知否》；侍妾人物 → 看《琅琊榜》秦般若；武侠 → 看胡金铨；女尊 → 看现有女尊剧的剧照
2. **找参考剧照**：从对标作品里挑 1-2 张视觉锚点剧照、给用户看、问"是这种感觉吗"
3. **用 AI 协助优化提示词**：把剧照特征拆成"构图/光线/服化道/色调"四维写进 prompt
4. **绝对禁止用明星脸**——只参考造型、不能直接用真人照片做角色资产或视频生成的参考。这是合规底线、违反会触发审核 + 法律风险

### 朝代基线（古装剧服化道）

如果是古装剧、按朝代定基线：

| 朝代 | 视觉特征 |
| --- | --- |
| **秦汉** | 素雅质朴、深衣、曲裾绕襟、线条简洁 |
| **唐代** | 齐胸襦裙、高髻丰腴、色彩明艳、华丽开放 |
| **宋代** | 褙子流行、窄袖素雅、发髻简约、清雅内敛 |
| **明代** | 袄裙马面裙、高髻繁复、织金刺绣、端庄华贵 |

题材偏向：

- 古偶剧 → 艳丽华贵
- 仙侠类 → 空灵梦幻
- 武侠类 → 潇洒飘逸
- 宅斗/古装权谋 → 以某历史朝代为大体原型

## 待补充的范式

可以扩展的：
- 杨德昌（台湾长镜头 + 都市冷调）
- 阿巴斯（伊朗远景 + 公路）
- Lynch（梦境 + 红窗帘 + 跳剪心理）
- 蔡明亮（极慢长镜头 + 都市边缘）
- Tarkovsky（湿润 + 自然 + 长镜头）
- 杜可风摄影风格（独立摘出来、和王家卫合作但有自己语汇）

用户跑过对应风格后、可以反馈"这一条加进去"。
