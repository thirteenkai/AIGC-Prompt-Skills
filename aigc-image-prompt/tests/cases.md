# aigc-image-prompt 评估集

本文件供 aigc-image-prompt skill 使用。**改 skill 后必跑回归**——避免改了 A 部分破坏 B 部分。

---

## 怎么用

每次大改 SKILL.md 或 references 后、新开一个对话、把每个测试用例的"输入"作为用户输入、让副导演响应、人工对照"通过标准"打勾。全过才能合并修改。

---

## 评估样例（5 个，覆盖三大场景的典型路径）

### 用例 1：场景 A 文生图、信息严重不足

**输入**：
> 一个男人在路上

**通过标准**：
- [ ] 副导演没有直接出 prompt
- [ ] 主动追问 2-4 个问题（不能全部都问）
- [ ] 追问用人话（不出现"景别/构图/光比"这类词）
- [ ] 追问必含"情绪/场景/光线/参考"中至少 3 个维度
- [ ] 不擅自加角色细节、剧情

### 用例 2：场景 A 文生图、信息充足 + 含情绪词

**输入**：
> 深夜便利店、冷清、年轻女孩、电影感、王家卫风格、孤独感

**通过标准**：
- [ ] 副导演触发情绪追问（"孤独"必须下沉到可见信号）
- [ ] 主动追问"想让观众通过什么具体动作 / 眼神 / 身体信号感受到孤独"
- [ ] 用户回答后才出最终 prompt
- [ ] 【画面分析】含每个专业词的人话注释
- [ ] 【最终提示词】里不出现"孤独"标签、只有动作 / 生理 / 姿态信号
- [ ] 触发了 director-styles.md（王家卫范式拆解）

### 用例 3：场景 B 单图改图、修改项模糊

**输入**（带一张图）：
> 把外套改红色、然后背景换成街道

**通过标准**：
- [ ] 副导演追问 3 件事：什么红 / 什么街道 / 哪些保留
- [ ] 不擅自决定红色细节
- [ ] 用户补充后输出三段式指令（保留锚点 / 修改指令 / 一致性约束）
- [ ] 【保留不变】段明确列出
- [ ] 没在【最终提示词】里加用户没提的元素

### 用例 4：场景 C 多图风格迁移（最易出错）

**输入**（带两张图）：
> @图1 是我的角色（白发少年黑白长袍立绘）、@图2 是编导要的画风（水彩淡彩）。把图2 的画风套到图1 上、服装姿势不变

**通过标准**：
- [ ] 副导演不再追问（场景 C 且分工清晰）
- [ ] 【最终提示词】≤4 句
- [ ] 没在 prompt 里描述图1 的服装/发型/姿势（图已表达）
- [ ] 没在 prompt 里堆水彩技法形容词（图2 已表达）
- [ ] 含纠偏护栏（弱化硬边勾线 / 降低高光 / 发丝蓬松等至少 1-2 项）
- [ ] 【保留不变】段明确
- [ ] 通过 SKILL.md 场景 C 自检清单 4 项

### 用例 5：触发"图文分工"原则

**输入**（带 3 张参考图）：
> @图1 是我女主、@图2 是画风参考、@图3 是场景。把女主放进图3 的场景、用图2 的画风

**通过标准**：
- [ ] 副导演识别为多图参考（场景 C 进阶）
- [ ] 明确文字角色（只讲迁移方向 + 锚点 + 护栏）
- [ ] 【最终提示词】不重复描述任何参考图的内容
- [ ] 总 prompt 长度极简（≤5 句）
