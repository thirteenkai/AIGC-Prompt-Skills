## 示例 3：单镜多动作链（不要预防性拆段）

**用户输入**（带一张干净首帧）：
角色玩手机时突然吸气、憋了一下、然后打喷嚏、最后愣住、约 12 秒

**正确响应**（单镜动作生成 + 多动作链信号、按"不预防性拆段"原则出单镜）：

确认一件事：

- 第一次吸气是"小吸"还是"明显倒吸一口冷气"那种？这决定模型对突发感的表达。

用户："明显倒吸的那种、像快打喷嚏的前奏"。

【镜头分析】
- 首帧：角色坐在沙发上、视线聚焦手机屏幕、姿态放松
- 动作设计：玩手机 → 倒吸气 → 憋住 → 打喷嚏 → 愣住
- 运镜：固定镜头 → 把全部表演权交给人物
- 节奏：前缓后突 → 前段平静，中段突变，末尾停顿
- 时长：12 秒（Seedance 2.0 单镜能扛，不拆段）
- 一致性锚点：人物面部特征、服装、沙发位置、手机
- 声音设计：室内低频环境音 + 手机轻触声 + 吸气 / 喷嚏 / 衣料摩擦
- 工具选择：Seedance 2.0（多动作链 + 表情变化、Seedance 强项）
- 判断与取舍：用户给了整体方案，按单镜出。如果跑出来"打喷嚏到愣住"那段不稳，可以从那里拆成两段后期接，但先按单镜跑。

【最终提示词】（只复制下面代码块）

```text
【素材锚定】
首帧是角色坐在客厅沙发上，视线聚焦手机屏幕，姿态放松。参考图职责：@图1 锁定人物外观、服装、沙发空间和初始构图。

【镜头与动作】
固定中景，平视，三脚架不动，全程不推不拉不摇。起手：角色玩手机，手指滑动，表情有轻微变化，整体姿态放松。突变：角色突然明显倒吸一口冷气，肩膀微微一耸，表情瞬间凝固。憋住：角色眉头微皱，嘴唇紧抿，身体僵住。爆发：角色打喷嚏，头部短促前倾，肩膀震动一下。收尾：角色愣住，缓慢抬眼看向前方，表情停在惊愕状态。

【场景光影】
延续首帧客厅自然室内光，沙发和手机位置保持稳定，背景不过度变化，画面不闪烁。

【声音与约束】
环境音：安静客厅低频底噪。关键音效：手机轻触声、明显倒吸气、短促喷嚏、衣料轻微摩擦。整段保持人物面部特征、服装、发型、沙发位置、手机稳定一致。不要出现字幕、不要出现 BGM。
```

【工具与参数建议】
- 推荐工具：即梦 Seedance 2.0（多动作链 + 表情变化、Seedance 强项）
- 推荐时长：12 秒
- 推荐模式：首帧
- 兜底：如果"打喷嚏到愣住"段不稳，可以从那段拆为独立 3 秒接

## 示例 4：单镜情绪表达（破防瞬间）

**用户输入**：
角色看到手机里某个消息后破防、5 秒

**正确响应**（含情绪词，必须触发情绪追问）：

破防之前你想让观众看到什么？

1. 想让观众知道角色看到了什么吗？（手机屏幕的内容入画 vs 不入画）
2. 破防的强度：是泪光打转但没流（克制版），还是真的开始哭（外放版）？

用户："手机屏幕不入画、克制版"。

【镜头分析】
- 首帧：角色坐在台阶上、低头看手机、表情还是平静
- 情绪设计：平静 → 眨眼停顿 → 眼眶泛红 → 手遮住下半张脸
- 运镜：固定镜头 + 末尾轻微推近 → 让情绪长出来
- 节奏：前慢后收紧
- 一致性锚点：人物面部、服装、台阶环境
- 声音设计：室外环境底噪 + 呼吸变浅 + 衣料摩擦
- 工具：Seedance 2.0（面部微表情 + 情绪过渡）
- 未写"破防"标签到最终 prompt，只写动作和生理变化。

【最终提示词】（只复制下面代码块）

```text
【素材锚定】
首帧是角色坐在户外石阶上，低头看手机，表情平静。参考图职责：@图1 锁定人物外观、服装、台阶环境和手机位置。

【镜头与动作】
固定中近景，平视，镜头在 5 秒内极缓慢推近一点点。角色先保持低头看手机的姿态，表情平静；随后眨眼一次，视线停在手机上没有立刻移开，眼眶开始泛红，鼻翼轻微通红；嘴角向下拉同时上抬，像哭出来前的肌肉变化；最后缓慢抬手覆盖住下半张脸，画面停在手遮住嘴的瞬间。

【场景光影】
户外台阶自然柔光稳定，背景保持轻微虚化，手机屏幕不展示具体内容，只作为光线和动作触发点。

【声音与约束】
环境音：远处城市低频底噪和轻微风声。关键音效：呼吸变浅、衣料轻微摩擦。整段保持人物面部特征、服装、发型、台阶环境稳定一致。不要出现字幕、不要出现 BGM。
```

【工具与参数建议】
- 推荐工具：即梦 Seedance 2.0（面部微表情 + 情绪过渡稳定性好）
- 推荐时长：5 秒
- 推荐模式：首帧

## 示例 5：简单 5 秒首帧动作（四块结构但不强行时间线）

**用户输入**：
让角色慢慢抬头看镜头、5 秒

【最终提示词】（只复制下面代码块）

```text
【素材锚定】
首帧是角色低头站在室内窗边，脸部被侧光照亮，身体保持安静。参考图职责：@图1 锁定人物外观、服装、室内窗边环境和初始构图。

【镜头与动作】
固定中近景，平视，三脚架不动。角色先保持低头姿态，轻微呼吸；随后缓慢抬头看向镜头，眼神逐渐聚焦，动作克制自然；画面停在角色正视镜头的瞬间。

【场景光影】
窗边柔和侧光保持稳定，背景轻微虚化，人物脸部亮暗关系不闪烁。

【声音与约束】
环境音：安静室内底噪和远处轻微城市声。关键音效：轻微呼吸和衣料摩擦。整段保持人物面部特征、服装、发型、构图稳定一致。不要出现字幕、不要出现 BGM。
```

## 示例 6：12-15 秒复杂动作链（Seedance 优先 + 节拍分段）

**用户输入**：
角色在走廊里边走边接电话，听到坏消息后停下，回头看门口，最后把手机慢慢放下，15 秒

【最终提示词】（只复制下面代码块）

```text
【素材锚定】
首帧是角色站在狭长室内走廊入口，右手拿手机贴近耳边，身体朝走廊深处。参考图职责：@图1 锁定人物外观、服装、走廊空间和初始构图。

【镜头与动作】
稳定跟拍中景，镜头在角色正前方缓慢后退，保持角色腰部以上入画。起手：角色边听电话边向前走，脚步平稳，眼神看向走廊深处。转折：角色听到坏消息，脚步明显变慢，嘴唇微微张开但没有说话，手指握紧手机。停顿：角色停下，肩膀下沉，缓慢回头看向身后的门口。收尾：角色把手机从耳边慢慢放下，视线仍停在门口方向，画面停在手机垂到胸口高度的瞬间。

【场景光影】
走廊顶部冷白灯形成连续透视线，门口一侧有较暗阴影，背景纵深稳定不跳变。

【声音与约束】
环境音：走廊空调低频声、远处轻微回声。关键音效：稳定脚步声、手机贴近耳边的轻微摩擦、停步时鞋底摩擦地面的细声。电话里的人声只保持模糊，不生成清晰台词。整段保持人物面部特征、服装、发型、手机、走廊空间稳定一致。不要出现字幕、不要出现 BGM。
```

【工具与参数建议】
- 推荐工具：即梦 Seedance 2.0（15 秒多动作链 + 表情过渡 + 手机道具操作）
- 推荐时长：15 秒
- 推荐模式：首帧
