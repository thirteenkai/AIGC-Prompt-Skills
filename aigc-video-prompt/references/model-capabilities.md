## 默认选型策略

公开版默认 **Seedance 2.0 优先**。原因不是排除其他模型，而是视频 prompt 的默认形态已经升级为导演四块结构，Seedance 更适合承接多模态参考、动作节拍、对白、环境音和多动作链。

但只要需求明显落在可灵强项，就直接推荐可灵：

- 精准推拉摇移、希区柯克变焦、首尾帧空间过渡
- 烟雾、流体、雨雪、火焰、自然元素的物理运动
- 写实大幅动作或需要更强物理连贯性的短动作

输出时不要说"默认一定用 Seedance"。应写成："默认推荐 Seedance；如果你这条更重视 X，可以改用可灵。"

## Seedance 2.0（即梦视频）

**版本号**：2.0
**最后实测日期**：2026-05 之前（来自 SKILL.md 旧文）
**状态**：仍在使用

### 单镜能力（关键能力上限）

| 维度 | 实测上限 | 备注 |
| --- | --- | --- |
| 单镜时长 | 15 秒 | 物理上限、再长扭曲率显著上升 |
| 连续动作链 | 5+ 拍 | 例："玩手机→吸入气体→憋气难受→打喷嚏→定格惊愕" |
| 景别切换 | 1-2 次 | 单镜内做"中景→特写"或"中景→近景→特写"可行 |
| 空间转换 | 1-2 次 | 连续运动可达的（如画框前→走廊深处） |
| 角色变装 | 可做 | 同一人物在单镜内换装 |
| 道具/场景渐变 | 可做 | 单镜内场景从 A 变到 B |
| 360 度运镜 | 可做 | 完整环绕一周 |
| 跟随运镜的视角转场 | 可做 | 视点跟随主体移动 |

### 表演与对白

- **可以直接生成环境音、动作音效和角色对白**——prompt 里默认写环境音 + 关键动作音效，不要默认全留剪映
- **多人对白错位写**——不要全部同时（避免音轨糊）
- 短句台词稳、长句和复杂台词建议剪映兜底
- 面部一致性和微表情过渡稍强（比可灵）

### 强项 / 弱项

- ✅ 角色表演、面部表情、对白、道具操作、多动作链
- ✅ 风格化处理（插画感、抽帧、卡通化）
- ✅ 升格慢动作响应较好
- ⚠️ 烟雾、流体、自然元素的物理感不如可灵
- ⚠️ 精准运镜响应不如可灵（推拉摇移术语不如可灵准）

### 推荐时长（按场景）

| 场景 | 推荐时长 | 备注 |
| --- | --- | --- |
| 简单微动作 | 5 秒 | 最稳 |
| 完整动作 + 微运镜 | 5 秒 | 默认 |
| 多动作链（3-5 拍） | 8-10 秒 | 单镜能扛 |
| 多动作链 + 景别切换 | 10-15 秒 | 用足上限 |
| 极长静止（罗伊·安德森式） | 10 秒 | 可以用满 |

## 可灵 3.0

**版本号**：3.0
**最后实测日期**：2026-05 之前（来自 SKILL.md 旧文）
**状态**：仍在使用

### 单镜能力

| 维度 | 实测上限 | 备注 |
| --- | --- | --- |
| 单镜时长 | 10 秒以内最稳 | 超过 10 秒扭曲率显著上升 |
| 连续动作链 | 不及 Seedance | 复杂角色动作链建议 ≤3 拍 |
| 精准运镜 | 强 | 对运镜术语（推/拉/摇/移/环绕/希区柯克变焦）响应准 |
| 烟雾/流体/光线 | 强 | 自然元素物理感最好 |
| 写实质感 | 强 | 写实度稍高于 Seedance |
| 首尾帧过渡 | 稳定性好 | 空间移动 / 镜头位置过渡可靠 |

### 强项 / 弱项

- ✅ 烟雾、流体、自然元素飘动、光线变化
- ✅ 精准运镜（推拉摇移、希区柯克变焦）
- ✅ 大幅动作（走路、奔跑、打斗）连贯性稍好（但仍需 ≤5 秒）
- ✅ 写实风格、首尾帧过渡
- ⚠️ 多动作链不如 Seedance
- ⚠️ 风格化（抽帧、插画感）不如 Seedance

### 推荐时长

| 场景 | 推荐时长 |
| --- | --- |
| 默认 | 5 秒 |
| 简单运镜 + 单一动作 | 5 秒 |
| 复杂运镜（如希区柯克变焦） | 5-8 秒 |
| 烟雾/流体/自然元素 | 5-10 秒 |
| 大幅动作 | 3-5 秒 |

| 烟雾 / 流体 / 自然元素 | 可灵 3.0 | 物理感好 |
| 大幅动作（跑/跳/打斗） | 可灵 3.0 | 连贯性稍好 |
| 首尾帧 / 空间移动 | 可灵 3.0 | 过渡稳定 |
| 写实风格 | 可灵 3.0 | 写实度稍高 |
