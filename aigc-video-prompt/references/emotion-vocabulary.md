# 情绪 → 可见动作语料库（视频版）

本文件供 aigc-video-prompt skill 使用。SKILL.md 纪律 10 提出的核心原则：模型不会演"她失落"，但能演"缓慢放下手机，指尖在屏幕边停了一秒才松开"。本文件是 8 维度的扩展查表，含视频独有的"时间轴展开"维度。

## 使用纪律

1. **挑 2-3 个维度，按时间顺序写**。视频和静图的区别：动作是有"起 → 转 → 收"的，要落到时间轴上。错："她紧张地转头"。对："她先有半秒的呼吸停顿 → 手指从杯沿离开 → 才缓慢转头 → 转完头停在那里"。
2. **维度词不进【最终提示词】**。"生理反应明显""社交习惯入画"是失败输出。要写出具体动作。
3. **节奏配情绪强度**。强烈情绪用快节奏 + 多动作链；克制情绪用慢节奏 + 单点动作 + 停顿。
4. **情绪在画面里要"被发现"，不是"被宣告"**。前 1-2 秒先让观众看到平常状态，再让小信号慢慢露出来。

---

## 8 维度速查定义（视频补充）

| 维度 | 视频里的展开方式 |
| --- | --- |
| **内心活动** | 动作停顿（拿起手机但没解锁、伸手又收回）、动作和目光不同步 |
| **生理反应** | 呼吸节奏变化（变快/变慢/停顿）、吞咽、瞳孔变化、太阳穴血管浮起 |
| **社交习惯** | 这个人物特有的小动作在镜头里出现一次（不要重复） |
| **表情** | 表情从 A 状态过渡到 B 状态（不是从开始就到位） |
| **眼神** | 视线轨迹（先看哪里、停多久、再看哪里、最后停在哪） |
| **语气** | 台词 + 喉部动作 + 呼吸节奏一起设计（Seedance 2.0 可直接出对白） |
| **心理代入** | 镜头视角设计（POV / 跟拍 / 偷看 / 上帝） |
| **情感具象化** | 道具的状态变化（咖啡从冒热气到凉了、灯从亮到灭） |

## 常见情绪 → 动作翻译表（含时序节奏）

### 1. 紧张

时间结构：平常 → 出现刺激 → 身体先反应 → 表情后跟上

具体写法：
- 前 1 秒：保持原有动作（看货架/喝咖啡/翻手机）
- 第 2 秒：呼吸轻微停顿、手部动作减慢
- 第 3 秒：眨眼频率变高、嘴角向内抿
- 末态：视线避开刺激源、下颌微微绷紧
- 推荐时长：5 秒
- 节奏：前慢后微加速

### 2. 愧疚

时间结构：直面 → 回避 → 收缩

具体写法：
- 前 1 秒：与对方对视
- 第 2 秒：视线先向下落 1 秒、再抬起但偏向对方膝盖位置
- 第 3 秒：肩膀微微下沉、嘴唇抿紧但不用力
- 第 4 秒：双手在身前轻握或藏到口袋里
- 推荐时长：4-5 秒
- 节奏：均匀慢

### 3. 思念

时间结构：动作停顿 → 视线越界 → 嘴唇微启

具体写法：
- 前 1.5 秒：日常动作（端杯、翻书、打字）
- 第 2 秒：动作明显减慢、最后定在半完成状态（杯子端在半空、书翻到一半停住）
- 第 3 秒：眼神从手中物件越过、望向画面深处或画外
- 第 4 秒：嘴唇微启但没发声、轻微的呼吸停顿
- 推荐时长：5 秒
- 节奏：缓慢均匀

### 4. 克制

时间结构：刺激到来 → 用力压住 → 维持

具体写法：
- 第 1 秒：受到情绪刺激（听到一句话/看到一条信息）
- 第 2 秒：深呼吸一次、肩膀刻意挺直
- 第 3 秒：下颌咬紧、嘴唇压成线、目光定在一个点
- 第 4-5 秒：维持这个状态，只有呼吸的轻微起伏
- 推荐时长：5 秒
- 节奏：前快后停（点燃→压制→静止）

### 5. 破防

时间结构：维持 → 一个触发点 → 防线塌

具体写法（最常见用法 5 秒）：
- 前 1 秒：保持克制状态（看着屏幕、听对方说话）
- 第 2 秒：眨眼一次、眼眶开始泛红
- 第 3 秒：嘴角向下拉同时上抬（典型哭脸前奏）、鼻翼通红
- 第 4 秒：手抬起来覆盖住嘴或眼睛
- 第 5 秒：定格在手遮住部分脸的瞬间
- 推荐时长：5 秒
- 节奏：前慢后加速、最后定格

### 6. 暧昧

时间结构：动作和目光的"错位"

具体写法：
- 前 1.5 秒：和对方对话/共同动作（喝咖啡、走路）
- 第 2 秒：视线偷瞥对方一眼、对方没注意
- 第 3 秒：被对方目光接触瞬间、立刻挪开视线、嘴角不自觉上扬
- 第 4 秒：脸颊潮红、低头掩饰
- 推荐时长：5 秒
- 节奏：前平后变（关键在"被发现的瞬间"那 0.3 秒）

### 7. 失落

时间结构：动作完成 → 没有下一步 → 凝固

具体写法：
- 前 2 秒：完成某个动作（挂电话、关门、放下手机）
- 第 3 秒：保持动作完成的姿态、不动
- 第 4 秒：肩膀缓慢下沉、视线从物件上飘走
- 第 5 秒：定格在凝固状态
- 推荐时长：5 秒
- 节奏：动 → 静 → 更静

### 8. 疲惫

时间结构：长时间未变的状态

具体写法：
- 整段：靠在椅背/墙上、半阖眼、不聚焦
- 微动作：偶尔深呼吸一次、缓慢眨眼
- 末态：可以加一个"刚发现天亮了"的轻微反应（视线移向窗户、停留 0.5 秒）
- 推荐时长：5 秒
- 节奏：极慢、几乎静态

### 9. 喜悦（克制版）

时间结构：信号出现 → 内化 → 不自觉外溢

具体写法：
- 前 1 秒：看到/听到喜讯（看手机/听到对方说话）
- 第 2 秒：表情先停滞 0.5 秒（信息进入）
- 第 3 秒：眼睛先笑（眼角细纹出现）
- 第 4 秒：嘴角上扬但没张嘴
- 第 5 秒：脸颊上提、整个面部柔和
- 推荐时长：5 秒
- 节奏：先停后笑（不要从一开始就笑）

### 10. 愤怒（隐忍版）

时间结构：触发 → 身体先反应 → 压住

具体写法：
- 第 1 秒：听到/看到刺激
- 第 2 秒：太阳穴血管浮起、下颌咬肌凸起、呼吸变深慢
- 第 3 秒：手指逐渐握紧、关节发白
- 第 4 秒：嘴唇紧闭成一线、瞳孔锁定对方
- 第 5 秒：维持，眨眼极少
- 推荐时长：5 秒
- 节奏：前快后冻结

### 11. 孤独

时间结构：环境给空间 → 主体的小动作

具体写法：
- 整段：人物在画面边缘 1/3、大片空白指向画外
- 微动作：呼吸轻、偶尔眨眼、双手垂在身侧或松松放在膝上
- 关键瞬间：可以加"电话铃响起但没接"或"远处有人走过但她没看"的细节
- 推荐时长：5-8 秒
- 节奏：极慢、有节制的微动

### 12. 决断（下定决心瞬间）

时间结构：游移 → 转变 → 行动

具体写法：
- 前 2 秒：游移状态（视线四处飘、动作犹豫）
- 第 3 秒：深呼吸一次、肩膀从松弛变挺直
- 第 4 秒：眼神从游移变聚焦、嘴唇从抿到松开
- 第 5 秒：开始具体动作（起身/伸手按键/拿起钥匙）
- 推荐时长：5 秒
- 节奏：慢 → 转折点 → 加速

## 非人物场景的 3 维度（空镜 / 静物 / 场景图）

视频空镜不只是"静止画面"，要有"时间在流动"的微动作。

### 一、空间留白 + 微动

| 用法 | 空间设计 | 微动加什么 |
| --- | --- | --- |
| 主体被稀释 | 主体 1/4 + 空白 3/4 | 空白处的微动（窗帘飘动、咖啡热气升起） |
| 主体被挤压 | 主体顶满画面 | 镜头几乎不动、只有压迫感的呼吸节奏 |
| 主体偏一侧指向空白 | 视线/动势指向空白方向 | 空白方向有缓慢出现的元素（光线变化、风吹过） |
| 主体偏一侧撞墙 | 视线指向画框 | 主体周围非常静、只有遥远的环境音暗示 |

### 二、光影方向 + 时间流动

视频里光线可以变化（白昼推移、灯光开关、云遮挡）。

| 时间结构 | 写法 | 适合 |
| --- | --- | --- |
| 静止光 | 整段光线不变、只有微动 | 凝固的情绪、长时间状态 |
| 光的轻微移动 | "夕阳光线从地板缓慢爬过墙面"、"窗帘的影子在墙上慢慢移动" | 时间感、思绪流动 |
| 光的开关 | "远处灯一盏盏熄灭"、"窗外天色从亮变暗" | 离别、结束、转折 |
| 突然光变 | "外面突然下雨、雨声响起、画面整体变暗" | 戏剧节点 |

### 三、物体的叙事暗示 + 状态变化

视频可以做"物体从 A 状态到 B 状态"。

| 物体起点 → 终点 | 暗示 |
| --- | --- |
| 咖啡冒热气 → 不冒了 | 时间过去了很久 / 等了很久没人来 |
| 桌上手机屏幕亮 → 自动熄灭 | 没收到回复 / 期待落空 |
| 香烟点燃 → 一根烧完 | 长时间的焦虑/等待 |
| 蜡烛点燃 → 风吹熄 | 命运、转折、结束 |
| 窗外路灯亮 → 关掉 → 天亮 | 一整夜未眠 |
| 桌上闹钟 → 响起 | 该出发了 / 时间到了 |

## 反例集

### ❌ 把情绪标签直接当 prompt

错：她紧张地转头看窗外。

正：女孩保持望货架的姿态约 1 秒、呼吸轻微停顿、手指从杯沿离开、然后缓慢转头望向窗外、转完头停在那里。

为什么改：把"紧张"拆成 3 个可见信号（呼吸停顿+手指离开+缓慢转头），并放到时间轴上。

### ❌ 维度堆叠 + 同时发生

错：第 2 秒她紧张地呼吸加快、手指摩挲杯沿、视线避开、嘴角抿紧、肩膀内收、太阳穴血管浮起。

为什么错：一个 0.5 秒的瞬间堆 6 个动作，模型会做成"全脸都在抽搐"。

正：第 2 秒呼吸轻微停顿、第 3 秒手指从杯沿松开、第 4 秒视线缓慢避开。把动作铺到不同秒上。

### ❌ 矛盾节奏

错：她突然紧张地缓慢转头。

为什么错："突然"是快、"缓慢"是慢，节奏冲突。

正：要么"突然惊觉一震、迅速转头"（突然=快）；要么"呼吸停顿、缓慢转头、停在转头完成的侧脸"（紧张=慢）。

### ❌ 情绪没"出现"过程

错：女孩崩溃大哭。

为什么错：直接从平静跳到大哭、AI 视频做不出。

正：女孩盯着手机屏幕约 1 秒、眨眼一次、眼眶开始泛红、嘴角向下拉、抬手覆盖住下半张脸、定格。情绪要"长出来"。
