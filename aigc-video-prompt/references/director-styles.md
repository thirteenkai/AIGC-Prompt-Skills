## 使用纪律

1. **不要在【最终提示词】里写"王家卫风格"**。把要素拆开写。
2. **范式不混搭**。一条 prompt 里同时用王家卫抽帧 + Wes Anderson 横移、模型迷茫。
3. **运镜要和情绪节奏匹配**。不要"是枝裕和的题材"配"诺兰的推镜"。
4. **图版的画面要素仍然适用**——本文件只补运镜和节奏。完整要素查图版 director-styles.md。

## 范式 1：王家卫

### 视频独有要素

- **抽帧感**（核心）：画面既有动态模糊（拖影）又有卡顿（不连续）、像旧 VHS 调慢回放
- 缓慢运镜、几乎不动
- 镜头里物理时间被拉长（慢动作 + 抽帧叠加）
- 长焦 + 浅景深的特性在视频里更明显（人脸有"漂浮"感）

### 推荐运镜

- 固定镜头 + 缓慢推近（5-8 秒、推 1/4 距离）
- 跟随侧移（相机和人物保持平行）
- 不用快速运镜、不用希区柯克变焦、不用甩镜

### 时间结构

- 8 秒最适合（足够酝酿）
- 节奏：极慢 → 末态定格
- 主体动作：1 个完整缓慢动作 + 大量微动作

### prompt 翻译模板（视频）

```
首帧是 [人物 + 场景 + 王家卫范式画面要素（参考图版）]。
缓慢推近镜头（约 8 秒推进 1/4 距离），轻微抽帧感（画面带拖影 + 偶尔卡顿）。

开场约 2 秒，人物保持原状，只有 [呼吸/眨眼/手指动作]。

接着，人物 [一个缓慢的核心动作，如"缓慢转头""缓慢抬手抚摸杯沿"]。

最后，定格在 [情绪状态的末态侧脸/局部]。

整段保持人物面部特征、[标志性细节]、冷暖光对冲色调、胶片颗粒质感稳定一致。不要出现字幕、BGM。
```

### 模型选择

Seedance 2.0 在抽帧 + 人物表演上稍好（参考 model-capabilities.md）。

## 范式 2：是枝裕和

### 视频独有要素

- 几乎静态、极少运镜
- 镜头长（8-10 秒）但内容简单
- 画面里有自然的微动（窗外树影、远处人物走过、风吹窗帘）
- 自然光的微变化（云遮挡、夕阳推移）

### 推荐运镜

- 固定镜头（最常用）
- 偶尔极缓的横移（10 秒移动 1/4 画面）
- 不用推拉、不用跟拍

### 时间结构

- 8 秒
- 节奏：极慢、平静
- 主体动作：1 个非戏剧性的日常动作（喝茶、翻书、看孩子）

### prompt 翻译模板

```
首帧是 [人物日常动作 + 室内自然光场景]。固定镜头，全程不推不拉不摇。

开场约 3 秒，人物 [日常动作进行中]，背景 [窗帘飘动/孩子在玩/光线缓慢爬过墙面]。

接着，人物 [一个轻微的转折——抬头看窗外/放下手中物件/微笑]。

最后，回到平静状态，画面[末态简短描述]。

整段保持人物面部特征、服装、自然光氛围、暖色调克制不张扬稳定一致。不要出现字幕、BGM。
```

## 范式 3：Wes Anderson

### 视频独有要素

- 横移（左右平移）+ 推拉、几乎不用其他运镜
- 严格匀速（不缓入缓出、不变速）
- 演员的动作"装饰性"（机械、整齐、不像真人）
- 切换镜头时常用"九十度转向"的剪辑节奏

### 推荐运镜

- 横移（左→右、匀速）
- 直推 / 直拉
- 不用环绕、不用希区柯克、不用手持

### 时间结构

- 5-8 秒
- 节奏：匀速、机械感
- 主体动作：定点动作（坐着、站着）+ 装饰性微动

### prompt 翻译模板

```
首帧是 [人物在正中央、严格对称构图、Wes Anderson 范式画面]。镜头从左向右匀速横移（约 5 秒移动 1/3 画面）。

人物保持正面或正侧面、姿态僵直、做 [一个机械整齐的微动作如"转头 90 度"]。

[整段保持的色板和细节]。整段保持严格对称构图、高饱和限定色板、画面扁平质感稳定一致。不要出现字幕、BGM。
```

## 范式 4：罗伊·安德森

### 视频独有要素

- 完全静止镜头
- 极长（8-10 秒上限、用足）
- 整段画面几乎没有动（只有偶尔的微动）
- 人物面无表情、动作极缓

### 推荐运镜

- 完全静止
- 不用任何运镜

### 时间结构

- 10 秒（用满）
- 节奏：停滞感
- 主体动作：几乎不动、偶尔有一个轻微动作（眨眼、转头一点点）

### prompt 翻译模板

```
首帧是 [极远景 + 巨大空间 + 渺小人物 + 罗伊·安德森范式画面]。完全静止镜头、约 10 秒。

整段几乎不动、人物保持原位姿态、表情木然。

约第 6 秒，[一个极小的动作——人物转头一点点 / 远处有人走过 / 风吹动窗帘]。

之后回到静止。整段保持冷调灰白色调、停滞感、面无表情稳定一致。不要出现字幕、BGM。
```

## 范式 5：诺兰

### 视频独有要素

- 缓慢精准推近（不晃、不抖）
- 大画幅感（宽屏视野）
- 时间错乱可视化（沙漏倒流、镜面反射出另一个时空）
- 物理感强（人物动作有重量）

### 推荐运镜

- 缓慢直推
- 升降（在大空间里）
- 不用手持、不用快速运镜

### 时间结构

- 5-8 秒
- 节奏：稳定、有压迫感
- 主体动作：1 个有"分量感"的动作（缓慢站起、伸手按下按钮）

### prompt 翻译模板

```
首帧是 [人物 + 开阔纵深空间 + 诺兰范式冷蓝调画面]。镜头缓慢直推（约 6 秒推进 1/3 距离）、稳定无抖。

开场约 2 秒，人物保持原姿态、画面纵深感强。

接着，人物 [一个有重量感的核心动作，如"缓慢起身""按下按钮""拿起一件物品"]。

最后，[末态镜头停在人物动作完成的瞬间]。

整段保持冷蓝调、高对比度、写实质感稳定一致。不要出现字幕、BGM。
```

## 范式 6：贾樟柯 / 侯孝贤

### 视频独有要素

- 固定长镜头（10 秒用满）
- 画面里有多层动作同时发生（前景一个人吃饭、中景一个人路过、远景一辆摩托）
- 自然光、阴天、低饱和
- 中国/东亚特有的环境质感

### 推荐运镜

- 完全固定
- 偶尔极缓横移
- 不用推拉、不用跟拍

### 时间结构

- 10 秒（用满）
- 节奏：自然流动、像在偷看真实生活
- 主体动作：日常完整动作 + 环境里其他元素的动作

### prompt 翻译模板

```
首帧是 [中国小镇/县城/城中村具体场景 + 人物日常动作]。完全固定镜头、约 10 秒。

人物 [完整日常动作，如"吃完一碗面""走完一段路""抽完一根烟"]。

同一画面里，[环境其他动作：远处有摩托驶过 / 邻居走出来倒水 / 风吹动招牌]。

整段保持阴天漫射光、低饱和粗粝质感、深景深稳定一致。不要出现字幕、BGM。
```

## 范式 7：David Fincher

### 视频独有要素

- 精确缓慢推近（比诺兰更微妙、像不易察觉的逼近）
- 镜头机械感强（像广告片）
- 画面信息精确到像素
- 配合戏剧化的微表情变化

### 推荐运镜

- 极缓推近（5 秒推 1/8 距离）
- 不用手持、不用快速运镜

### 时间结构

- 5-8 秒
- 节奏：缓慢、压迫
- 主体动作：克制的小动作 + 微表情变化

### prompt 翻译模板

```
首帧是 [人物 + 单点硬光源 + Fincher 范式低饱和冷绿调画面]。镜头极缓推近（约 6 秒推进 1/10 距离）、稳定无抖。

整段人物保持基本姿态、做 [一个克制的小动作如"轻轻转头""目光从画面右侧移到正前方"]。

[微表情变化：从平静到察觉/到下决心/到察觉危险]。

整段保持极低饱和冷绿调、暗部扎实、单点硬光稳定一致。不要出现字幕、BGM。
```

## 范式 8：奉俊昊

### 视频独有要素

- 同一画面多层动作（前景一件事、后景另一件事同时发生）
- 镜头位置稳定、让多个动作"自己讲故事"
- 信息密度高、需要观众"读画面"

### 推荐运镜

- 固定镜头（首选）
- 偶尔缓慢横移
- 不用复杂运镜（运镜复杂会让多层动作糊掉）

### 时间结构

- 8-10 秒（用足让多层动作展开）
- 节奏：自然真实
- 主体动作：前景人物完整动作 + 后景另一个独立动作

### prompt 翻译模板

```
首帧是 [前景 + 中景 + 后景多层空间画面，奉俊昊范式]。固定镜头、约 8 秒。

前景：[主人物完整动作]。
中景或后景同时：[次要人物动作或环境动作，暗示对比关系如阶级/年代/生死]。

整段保持冷暖混合光、写实但戏剧化一档、信息密度高稳定一致。不要出现字幕、BGM。
```

## 多镜头组合参考

不是导演风格、是节奏组合提示：

- **王家卫式开场 → 是枝裕和式中段 → 罗伊·安德森式结尾**：情绪从浓到淡到空
- **Wes Anderson 全片**：每个镜头都对称、整片视觉统一
- **奉俊昊式整片**：每个镜头都信息密度高、多层叙事

## 待补充的范式

视频导演还可以扩：
- 杨德昌（台湾长镜头 + 缓慢横移）
- Tarkovsky（极长静止 + 湿润自然）
- 蔡明亮（10+ 秒静止 + 都市边缘）
- 黑泽明（动作场面的多机位剪辑）
- Christopher Doyle 摄影风格（独立摘出来）

用户跑过对应风格后、可以反馈"这一条加进去"。
