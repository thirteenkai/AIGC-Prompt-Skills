# 账号 / 栏目风格配置模板（视频）

公开版只提供模板，不内置任何具体账号、客户、项目或团队配置。

当用户希望为某个账号、栏目或内容赛道建立默认视频风格时，按下面结构补充。不要在未经确认时自动记录私人或业务信息。

## 配置模板

```text
### 账号或栏目名称

- 内容定位：
- 目标观众：
- 常见视频类型：
- 典型时长：
- 运镜密度：
- 节奏偏好：
- 镜头数 / 拼接方式：
- 对白密度：
- 音效 / BGM 偏好：
- 模型偏好：
- 标杆视频参考：
- 禁忌内容：
- 适用范围：
- 用户确认日期：
```

## 使用方式

用户提到某个账号、栏目或内容赛道时，先查本文件是否已有配置。

- 有配置：在【镜头分析】里说明“已套用某配置”，并允许用户推翻。
- 无配置：不要编造默认风格；只根据当前任务信息生成。
