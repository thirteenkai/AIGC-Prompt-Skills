# 视频提示词结构控制法

本文档供 aigc-video-prompt skill 使用。作用是把视频 prompt 从"描述画面"升级为"导演控制书"。默认用于最终提示词的四块结构：素材锚定、镜头与动作、场景光影、声音与约束。

---

## 一、导演四块结构

### 1. 素材锚定

先说明每个参考素材负责什么，避免模型乱用参考图。

可写职责：

- 角色身份：脸、发型、体态、标志性配饰
- 服装造型：衣服、鞋包、道具佩戴方式
- 场景空间：房间布局、街道方向、舞台位置、门窗关系
- 色彩 / 光线：色卡、主色调、阴影方向、特殊光效
- 声音参考：台词音色、语速、口音、情绪强度
- 首帧 / 尾帧：只锁起点或终点，不让模型把两张图混成一张

纪律：

- 参考图里已经有的视觉细节不重复描述，只写"@图N 锁定..."。
- 每个 @图N 第一次出现时说明职责。
- 多角色时每个角色独立标注，不用"两个人"这种模糊称呼。

### 2. 镜头与动作

这一块负责"怎么拍"和"发生什么"。

基础顺序：

1. 机位：固定 / 跟拍 / 平移 / 推近 / 俯拍 / 仰拍
2. 景别：特写 / 中近景 / 中景 / 全景 / 远景
3. 主体动作：谁，从什么状态到什么状态
4. 入画 / 出画：从哪边入，往哪边离开，是否被遮挡
5. 节拍：普通镜头自然描述，复杂镜头先分起手 / 中段 / 收手或分镜段落；只有存在明确时间锚点时才写秒级时间线

普通 5 秒单动作：

```text
固定中近景，平视，三脚架不动。角色先保持原姿态，轻微呼吸和眨眼；随后缓慢转头看向画面右侧，动作自然不急促；画面停在转头完成的侧脸。
```

复杂 12-15 秒动作链，如果时间分配明确：

```text
0-3 秒：固定中景，角色保持原姿态，手指轻微动作。
3-7 秒：角色抬头，视线从道具移向门口，肩膀慢慢绷紧。
7-12 秒：镜头轻微跟随，角色起身向门口走去，手始终握住道具。
12-15 秒：角色在门口停住，半回头，画面定在停顿瞬间。
```

复杂 12-15 秒动作链，如果时间分配不明确：

```text
固定中景，角色保持原姿态，手指轻微动作。
起手：角色抬头，视线从道具移向门口，肩膀慢慢绷紧。
中段：角色起身向门口走去，手始终握住道具，镜头轻微跟随。
收手：角色在门口停住，半回头，画面定在停顿瞬间。
```

### 3. 场景光影

光影是画面质量杠杆，不是装饰词。

优先写：

- 主光源从哪里来
- 阴影在画面哪一侧
- 背景是清晰还是虚化
- 色调是否稳定
- 材质质感是否需要锁定
- 是否有空气尘埃、雨、雾、反光等时间变化

少写：

- 电影级、大片感、高级感、震撼、绝美
- 超长导演名堆叠
- 和动作无关的装饰性风格词

### 4. 声音与约束

默认写声音层，但默认不写 BGM。

声音层顺序：

1. 环境音：空间底噪，例如空调、街道、风、雨、远处人声
2. 关键动作音效：脚步、衣料、门、道具、碰撞、呼吸
3. 对白 / 旁白：只有用户需要时写，短句优先
4. 静默：用于转折、停顿、突然断声

约束层顺序：

1. 身份一致：脸、发型、服装、标志细节
2. 空间一致：场景位置、门窗、道具方向
3. 机位一致：固定就不要推拉摇移，跟拍就不要突然切镜
4. 节奏一致：正常速度 / 慢动作 / 不要抽帧
5. 输出禁项：不要字幕、不要 BGM、水印等

## 二、何时加节拍或时间线

先区分三件事：

- **自然动作链**：一句话按顺序写，让模型自己分配节奏。
- **分镜节拍**：用"分镜一 / 分镜二"或"起手 / 制作 / 反应 / 收尾"写动作顺序，不写秒数。
- **秒级时间线**：写 0-3 秒、3-5 秒这类时间窗口，只在需要精确控制时使用。

### 不加时间线

- 5 秒以内的单一动作
- 首帧只需要"动起来"
- 情绪微动作
- 产品 / 道具轻微展示
- 用户希望模型自然发挥节奏

### 加分镜节拍，不加秒数

- 10 秒以上多动作链，但没有明确卡点
- 多角色短对白，但只需要错开台词和动作
- 制作过程、表演过程、反应过程需要顺序清楚，但不需要精确到秒
- 时间分配拿不准，硬写秒数会误导模型

### 加秒级时间线

- 用户明确给出每段时长或要求卡点
- 硬切、遮挡转场、擦镜、变装、入画 / 出画需要发生在明确时间点
- 音效、台词、动作要精确同步
- 尾帧要给下一段做精确衔接
- 强叙事段落里每一拍的时间窗口很清楚

### 不要因为这些原因加秒数

- 只是动作多
- 只是对白多
- 只是视频是 15 秒
- 只是担心模型漏动作

这类情况用分镜节拍更稳。秒数没有把握时，应该把控制力放在动作顺序、入画出画、道具状态、声音层和稳定约束上。

### 可能需要秒级或节拍二选一

- 多角色对白或正反打：如果需要严格正反打，写秒级；如果只是同一机位内对话，写分镜节拍
- 强叙事段落：如果有硬切 / 遮挡 / 音效同步，写秒级；如果只是情绪递进，写分镜节拍
- 动作中有关键遮挡 / 变装 / 转场：如果遮挡必须卡点，写秒级；如果只是顺序事件，写分镜节拍

## 三、常见控制模块

### 入画 / 出画

需要写清：

- 从画面左 / 右 / 下方 / 前景遮挡处入画
- 从哪一侧离开
- 是走出画、被遮挡、还是镜头转走

### 人物比例

人物比例影响模型是否跑偏。

示例：

```text
角色保持腰部以上中近景，占画面竖向约 2/3，道具位于画面下方 1/3，脸和道具都清晰可见。
```

### 尾帧锚定

用于连续视频。

```text
尾帧定格在角色右手握住门把、身体半侧向门口、头部还未完全转过去的瞬间，方便下一段接同一动作。
```

### 反向锚定

当模型容易做错时，先写正向画面，再写必要禁项。

```text
固定镜头，三脚架不动，只允许主体移动。不要推近、不要横移、不要环绕、不要切镜。
```

## 四、公开版写作纪律

- 不写参考资料来源。
- 不写历史案例、账号策略或素材路径。
- 不把某个具体项目的角色名、素材 ID、台词原文沉淀为通用规则。
- 可以沉淀的是结构、判断规则、约束顺序和输出格式。
