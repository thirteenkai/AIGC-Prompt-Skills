# 分镜设计 tips（多镜头串联用）

本文件是给 **AI 分镜师做"一系列分镜"时**的设计参考，**不是单镜 prompt 模板**。
单镜 prompt 写法看 SKILL.md 和 video-vocabulary.md。

用户的真实工作流：编导给剧本 → 用户拆成 N 个分镜 → 每个分镜先用 aigc-image-prompt 出图 → 再用 aigc-video-prompt 转视频 → 最后剪辑串联。
**每个分镜独立生成**，所以"前后镜头怎么衔接"的设计必须在拆分镜阶段就想好，AI 不会帮你串。

---

## 一、相邻分镜串联：相似元素法

世间所有"流畅转场"的底层逻辑，要么是**相似**，要么是**遮挡**。前期分镜阶段就预留好元素，后期才剪得动。

### 相似元素的 5 种设计

| 类型 | 怎么设计 | 经典案例 |
| --- | --- | --- |
| **形状相似** | 前镜结尾的物体形状 = 后镜开头的物体形状 | 《2001 太空漫游》骨头抛向空中 → 形状相似的太空飞船 |
| **动作相似** | 前镜人物做某动作未完成 → 后镜人物完成相同/承接动作 | 抛杯子未落 → 仰头看飞起的帽子 / 跳水落入水面 |
| **声音相似** | 前镜末尾出现的声音 → 后镜开头延续相同声音 | 警报声 / 钟声 / 雨声 |
| **颜色相似** | 前镜结尾主色调 = 后镜开头主色调 | 红色衣服 → 红色霓虹 → 红色鲜血 |
| **运镜相似** | 前镜末尾的甩镜方向 = 后镜开头的甩镜方向 | 镜头快速向左甩 → 顺势向左甩入下个画面 |

### 设计建议

拆分镜时，**每两个相邻分镜至少预留一个相似锚点**。如果完全没有相似元素，就在中间补一个**空镜过渡**（见下条）。

---

## 二、空镜过渡：解决"跳点"

人物从场景 A 突然出现在场景 B，没有中间画面会"跳"。在两个剧情镜头之间补一个**空镜头**作为过渡：

- 人物走廊行走的背影
- 推门 / 上下楼梯
- 环境特写（窗外的雨、桌面的咖啡、墙上的钟）
- 主角的脚走向某个方向

**空镜可以用 aigc-image-prompt 单独出图、aigc-video-prompt 加微动作或镜头平移。**

---

## 三、遮挡转场（蒙版换装类）：分镜阶段就要锁机位

如果想做"路人/汽车划过 → 主角变装"这种隐藏切点：

1. **机位绝对固定**：两个分镜的机位、焦段、构图必须完全一致
2. **预留遮挡物**：前后两个分镜里都要有同一个"划过的物体"作为切点（车、柱子、行人）
3. **运动轨迹一致**：人物如果在走，两次走的速度、路线、节奏要尽量对齐
4. **后期剪辑**：以遮挡物边缘为蒙版擦除上层

**AI 分镜的复刻方式**：用 aigc-image-prompt 锁住相同的构图和光线参数（参考 aigc-image-prompt 场景 C 的"保留锚点"写法），换装/换背景靠场景 B 改图实现。两段独立生成的视频再后期合成。

---

## 四、声音先行：让切点丝滑

下一个镜头的关键声音 **提前 0.5-1 秒** 在当前画面结尾出现。视觉切换的割裂感会被听觉引导掩盖。

**写 prompt 时**：在当前镜头末尾加一句"末尾 0.5 秒响起 [下个场景的声音]"。Seedance 2.0 能直接生成 prompt 里写的音效。

---

## 五、轴线一致性（多镜对话戏）

两个人对话的虚拟连线 = 轴线。**多个分镜全都要在轴线同一侧拍**，否则视线和位置会反掉，观众错乱。

**AI 分镜的实操**：
- 用 aigc-image-prompt 出对话戏首帧时，明确写"人物 A 在画面左、面朝右；人物 B 在画面右、面朝左"
- 后续分镜继续锁定这个左右关系
- **越轴只在一种情况下用**：刻意制造冲突/精神崩溃（参考《无耻混蛋》农舍戏）。这种时候 prompt 里直接写"反向机位、人物左右关系翻转"

## 六、组合运镜公式（多镜模板）

下面这些是把多种基础运镜串在一起的"段落级模板"，可以拆成 2-3 个分镜实现：

| 公式 | 拆成分镜 |
| --- | --- |
| **进入主体三段式** | 镜头 1：远景固定 → 镜头 2：跟随推近 → 镜头 3：人物特写定格 |
| **释放孤独三段式** | 镜头 1：人物近景 → 镜头 2：缓拉到中景 → 镜头 3：拉远到远景，人物变小 |
| **英雄登场公式（迈克尔·贝）** | 镜头 1：环绕走位、主角蹲下未入画 → 镜头 2：主角猛起立入画、背景烟雾 → 镜头 3：仰拍特写定格 |
| **推镜+变焦双重逼近** | 镜头 1：中景对话 → 镜头 2：缓推 + 变焦推近脸部 → 镜头 3：大特写定格 |
| **首尾呼应** | 第一个分镜与最后一个分镜画面/构图/动作呼应（参考《阿甘正传》开头羽毛降落 / 结尾羽毛升空） |

---

## 七、影片视角的统一性

整段视频如果一会儿用"跟随视角"一会儿用"上帝视角"，观众会迷失。**一段视频选定一个主导视角**，其他视角作为点缀（如某个关键瞬间切窥探视角制造悬疑）。

视角分类参考 video-vocabulary.md「视点视角」一节。

---

## 八、节奏设计（多镜的快慢配比）

不要全部都用同一节奏：

| 段落 | 推荐节奏 | 单镜时长建议 |
| --- | --- | --- |
| 开场 | 慢节奏 + 远景 + 推镜 | 5-8 秒 |
| 铺垫 | 中节奏 + 中景 + 微动作 | 3-5 秒 |
| 高潮 | 快节奏 + 多角度 + 碎剪 | 单镜 2-3 秒 + 多镜串联 |
| 情绪释放 | 慢节奏 + 拉镜 + 定格 | 5-8 秒 |
| 收尾 | 慢节奏 + 升格 / 抽帧 / 静帧 | 5-10 秒 |

---

## 九、人物塑造（不靠台词立人设）

如果分镜里出现"有身份/讲究"的人物且不便用对白：

- 第一个分镜：背影或局部，不给正脸
- 第二个分镜：特写袖扣 / 领带 / 手表 / 徽章 / 手势
- 第三个分镜：背影跟拍他做象征性动作（给小费、签字、握手）
- 揭示正脸的分镜留到最后

**情绪戏的"藏与放"**：高潮释放前一个分镜只拍背影或腿部（藏住表情），下一个分镜直接切进面部痛哭（瞬间释放），不要中间过渡。

## 十、长台词与对话戏的拆分（实战经验值）

### 单镜台词长度上限

- **15 秒单镜内塞 > 20 字台词、模型会嘴瓢/改词/吞词**——模型为了在视频时长内念完台词会强行加速到 rap 速度
- 长台词的演员表演也会丢失（变成"念词"而不是"演")
- 经验阈值：每 10 字台词约对应 2 秒视频时长
- 这条在做分镜脚本时就要确定、不要把超长台词带到 aigc-video-prompt 阶段

### 拆分策略

把单镜长台词拆成 2-3 个独立分镜：

| 原始设计（单镜 15s 塞超长台词） | 拆分后 |
| --- | --- |
| 长辈说一大段 + 晚辈一个反应（单镜 15 秒）| 镜头 1：长辈说前半段（5-7 秒）。镜头 2：晚辈反应（2-3 秒）。镜头 3：长辈说后半段（5-7 秒）|
| 多人对话集中 | 用"反应镜头机制"——长台词进行时、切到听众的反应镜头（震惊/贪婪/惊恐特写）。原台词不断、标注为说话人的画外音（O.S.）|

### 转头/转身的衔接补救

拆段后两镜的衔接动作（如转头中间）不流畅：

1. 截 A 镜头转头开始那一帧 + B 镜头转头完成那一帧
2. 用首尾帧生视频生成中间过渡（约 2-3 秒）
3. 在剪辑里把这段过渡插在 A 和 B 之间

## 十一、自检清单（一系列分镜出完前过一遍）

- [ ] 每两个相邻分镜有没有至少一个相似锚点？没有就补空镜
- [ ] 对话戏的轴线一致吗？人物左右关系有没有反过来？
- [ ] 关键道具/服装/光线有没有在分镜之间漂移？需要在每个 prompt 里写明保留
- [ ] 影片视角统一吗？还是乱跳？
- [ ] 节奏快慢配比合理吗？还是全程一个速度？
- [ ] 开场和结尾有没有呼应？
- [ ] 高潮处的情绪戏，有没有"藏"再"放"的设计，还是平铺直叙？
