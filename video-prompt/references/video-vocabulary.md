## ⚠️ 运镜三大黄金法则（违反直接崩）

来自火山引擎 Seedance 2.0 官方文档，是单条提示词里最容易翻车的三个点。写 prompt 前先过一遍。

### 法则 1：一次只用一个主要镜头指令

同一段提示词里堆 2 个以上运镜，模型会卡顿、抖动、回退到默认。

- ❌ "推镜+环绕+快速切换"
- ❌ "稳定特写+手持+无人机+环绕"（四种互斥运动堆一起）
- ✅ 一段时间用一种运镜；需要多种就拆段或用时间标记分配

复合运镜的正确写法是 **"Start: X. Then: Y."** 序列结构（不是 "X while Y" 同时句），且最多 3 个指令，2 个为理想：
- ✅ "Start: 缓慢推近建立场景。Then: 最后 2 秒向右轻摇。"
- ❌ "推近的同时向右摇并环绕"

### 法则 2：用节奏词，不要用摄影技术参数

模型听得懂 slow / smooth / gentle / gradual，听不懂 24fps / f/2.8 / ISO 800 / 85mm 焦距——后者会被忽略甚至导致混乱。

- ❌ "24fps, f/2.8, ISO 800, 焦距 85mm 拍摄"
- ✅ "缓慢平滑推近"
- 像跟剪辑师说话一样描述节奏，不要像跟工程师对接参数

### 法则 3：相机移动和主体移动必须分开描述

官方把这条定性为"最常被忽略的错误"。把"相机转 + 主体也在转"混在一句里，模型不知道谁围着谁动。

- ❌ "镜头围绕跳舞的人旋转"（spinning camera around dancing person）
- ✅ "舞者缓慢旋转。相机保持固定取景。"

主体正在做复杂动作（奔跑、跳跃、转身）时，相机指令应尽可能简洁（固定 / 轻微跟拍），避免与主体运动干扰。

**记忆口诀**：一个运镜、节奏词、相机和主体分开写。

---

### ⚠️ fast 是高风险词，单独用会让模型混乱

未限定的 "fast" / "快速" 会让模型整体混乱——它分不清是相机快、主体快、还是动作快。需要"快"的感觉时改用：

- **具体动词**：sprinting / vaulting / dashing / 急速冲刺 / 猛然回头
- **明确指向**："quick whip pan"（明确是相机摇）、"角色快速转身"（明确是角色动）
- **升格反衬**："以慢动作呈现"——用慢凸显速度感（影视里 90% 的"动作戏"靠慢动作传递速度）
- **视觉模糊**："画面边缘运动模糊"——用画面语言传达速度，不写"快"字

---

### 中文优于英文的提示词术语

Seedance 2.0 在专业术语识别上**中文优于英文**。"俯拍 / 跟拍 / 环绕"比 top-down / tracking / orbit 更稳；"复古胶片风、暖色调、颗粒感明显"比 vintage film style, warm tones, obvious grain 出来的画面色调更准。

- 默认纯中文写
- 中英混写适合：动作描述用英文（动词粒度更细）+ 氛围/情绪用中文
- 警惕生僻词和方言——Seedance 识别度低

## 运镜（镜头运动 / Camera Movement）

**用户最陌生的一块，必须配人话。**

### 基础运镜（常用）

| 专业词 | 人话解释 | 效果与适用 |
| --- | --- | --- |
| 固定镜头 (static) | 相机完全不动，只有画面里的东西在动 | 最稳、让观众集中注意主体、AI 稳定性最高 |
| 推镜 (push in) | 相机慢慢向主体靠近（**机位真的位移**，不是变焦放大） | 情感拉近、紧张聚焦、发现感、影片开场常用 |
| 拉镜 (pull out) | 相机慢慢从主体后退 | 交代环境、情感抽离、孤独感、影片结尾常用 |
| 摇镜 (pan) | 相机原地水平或垂直转动 | 环视环境、跟随视线、揭示信息 |
| 移镜 / 横移 (truck) | 相机水平方向整体平移 | 平稳跟随、空间叙事 |
| 跟镜 (follow) | 相机跟着主体一起移动 | 沉浸感、代入感 |
| 升降镜 (crane) | 相机上下垂直移动 | 戏剧进场/退场、情绪起落、首尾呼应 |
| 手持晃动 (handheld) | 相机有轻微自然晃动 | 真实感、纪录片感、紧张感 |
| 甩镜 (whip pan) | 相机快速摆动 | 转场、速度感、冲击力 |

### 推/拉的两种本质（容易写混）

| 写法 | 实际机制 | 视觉差异 |
| --- | --- | --- |
| 机位推（dolly in） | 相机物理向前移动 | 空间透视改变（背景变化明显） |
| 镜头推（zoom in） | 相机不动，焦距放大 | 单纯放大，透视不变（背景平进） |

写 prompt 时如果要"电影感的进入"用"相机缓慢向人物移动"；如果要"复古武侠/邵氏感"用"快速变焦放大"。

### 环绕 vs 旋转（务必区分，是两件事）

| 名称 | 机制 | 怎么写 |
| --- | --- | --- |
| **环绕** (orbit) | 机位围绕主体做圆周运动（脚步走） | "相机围绕人物水平环绕，主体始终在画面中心" |
| **旋转** (roll) | 机位原地不动，镜头自身沿 Roll 轴翻转 | "镜头沿水平轴翻转 180 度" |

**迈克尔·贝式英雄环绕**：演员先蹲下不入画 → 相机开始环绕走位 → 演员中途猛起立入画 → 配合背景烟雾/光影。是塑造"英雄登场"的固定公式。

**垂直环绕**：相机沿垂直弧线从上方俯拍滑到下方仰拍，常用于婚礼抛捧花、武侠落地等高情绪瞬间。

### 进阶运镜（特殊技法）

| 专业词 | 人话解释 | 适用场景 / AI 稳定性 |
| --- | --- | --- |
| **希区柯克变焦 / 滑动变焦 / Vertigo** | 机位向后退 + 镜头同时变焦推进（或反之），主体大小不变、背景透视暴变 | 眩晕、紧张、灵感迸发、空间扭曲。**必须有纵深背景**（走廊、林荫道），纯白墙拍不出效果。可灵 3.0 响应较好但不稳，建议跑多遍 |
| **快速变焦 / 邵氏大变焦** | 极速变焦放大或缩小（1 秒内完成） | 复古武侠/邵氏港片爆发感、肾上腺素瞬间 |
| **POV / 第一人称视角** | 相机模拟主体的眼睛 | 沉浸感、代入感、悬疑惊悚 |
| **后备箱视角** | 相机置于狭小封闭空间内（柜子/后备箱/冰箱）从内向外仰拍 | 强烈视觉冲击、打破常规、昆汀·塔伦蒂诺式风格 |
| **打破第四堵墙** | 演员直视镜头说话/做动作 | 伪纪录片感、观众强制参与、戏剧张力 |

### 运镜速度

| 词 | 人话解释 |
| --- | --- |
| 缓推 / 慢推 | 3 秒以上完成的推镜，情感酝酿 |
| 快推 | 1-2 秒内完成，戏剧冲击 |
| 定点 | 镜头运动到某个位置停住 |
| 匀速 | 均速运动，稳 |
| 缓入缓出 | 两端慢中间快，自然 |

### 手持晃动的"呼吸感"三句口诀（不是瞎抖）

手持不是随便晃，幅度和频率必须配合人物状态：

1. **呼吸贴角色心跳**：平静时画面起伏微弱，紧张时起伏加大
2. **摇晃随情绪轻重**：醉酒/眩晕时晃幅大、追逐时颠簸感强
3. **变焦跟肾上腺素**：突发状况或情绪高潮时配合快速变焦

写 prompt 时可以这样标："轻微手持感、随主角呼吸微微起伏" / "剧烈手持晃动、跟随追逐节奏"。

### 组合运镜公式（多技法叠加，做"高级运镜"参考）

注意：单镜里同时叠加 3 种以上运镜，AI 稳定性会显著下降。建议挑 1-2 种主导，其他作为微动。

| 公式 | 适合场景 |
| --- | --- |
| 旋转 + 前推 + 上摇 | 主角登场、震撼揭示 |
| 平移 + 拉出 | 揭示空间、人物从环境中孤立 |
| 平移 + 变焦 | 跟拍人物 + 强化主体 |
| 推镜 + 变焦 | 双重逼近，情绪压迫 |
| 希区柯克 + 旋转 | 极致眩晕、精神扭曲 |
| 跟随推 + 环绕 | 主角光环、英雄出场 |

## 景别（Shot Size，画面里装多少）

景别决定观众与主体的**心理距离**。Seedance 2.0 直接理解中文术语，中文识别度高于英文。

| 景别 | 英文 | 画面范围 | 叙事功能 | 典型用法 |
| --- | --- | --- | --- | --- |
| **极特写** | Extreme Close-up (ECU) | 眼睛 / 嘴唇 / 局部细节 | 纹理细节、强烈情绪、产品聚焦 | 瞳孔变化、戒指特写、品牌 Logo |
| **特写** | Close-up (CU) | 面部填满画面 | 情绪表达、亲密感 | 角色情绪高点、流泪一刻 |
| **中近景** | Medium Close-up (MCU) | 头部到肩膀 | 对话、细微表情 | 双人对话场景 |
| **中景** | Medium Shot (MS) | 腰部以上 | 标准对话、动作展示 | 口播、对话、基础动作 |
| **全景** | Full Shot | 全身入画 | 完整动作、姿态、肢体语言 | 舞蹈、跑步、武打 |
| **远景 / 广角** | Wide Shot / Establishing | 完整环境 + 主体 | 场景建立、宏大感 | 视频开场定场镜头 |
| **过肩镜头** | Over-the-shoulder (OTS) | 从某人肩膀后看向另一人 | 对话锚定、视角切换 | 对话戏标准镜头 |

**经典景别 + 运镜组合公式**（专业级提示词的标志）：

| 公式 | 效果 | 场景 |
| --- | --- | --- |
| 中景缓推到特写 | 情感聚焦 | 角色情绪进入高点（**最常用**） |
| 特写拉到远景 | 揭示环境 / 戏剧反讽 | 视频结尾，把角色放回宏大环境里 |
| 过肩中景 → 反打过肩中景 → 双人中景 | 对话戏标准三镜调度 | 双人对话 |
| 低角度中景 | 力量感 | 反派登场、品牌权威片 |
| 跟拍背影中景 | 沉浸感、跟随感 | 走路戏 |

**冲突组合（避免）**：
- ❌ "广角微距"（广角和微距矛盾）
- ❌ "稳定特写 + 手持 + 无人机 + 环绕"（多种互斥运动堆一起）

## 拍摄角度（Camera Angle，相机从哪看）

角度决定观众和主体之间的**权力关系**和心理位置。

| 角度 | 英文 | 心理感受 | 用在哪 |
| --- | --- | --- | --- |
| **平视** | Eye level | 中性、自然 | 标准对话、日常场景（**默认**） |
| **低角度（仰拍）** | Low angle | 力量感、威胁感、英雄感 | 反派登场、品牌权威片、英雄定调 |
| **高角度（俯拍）** | High angle | 脆弱感、渺小感、服从感 | 表现角色弱势、客观观察 |
| **荷兰角 / 斜角** | Dutch angle | 不安、失衡、心理扭曲 | 惊悚片、心理戏、黑暗场景 |
| **鸟瞰 / 俯视** | Bird's eye / Overhead | 全知视角、棋局感 | 规划场景、追踪、宏观布局 |
| **POV 主观视角** | First-person view | 沉浸感、生理代入 | 悬疑、惊悚、跑酷、极限运动 |

### POV 主观视角的特殊难度（AI 视频里最难做）

直接写"POV 视角"模型常常漂回"第三人称观察相机"。必须 6 个要素同时落地才稳：

1. **命名相机身份**：iPhone 拍摄 / GoPro 拍摄 / 行车记录仪 / 头戴运动相机
2. **画面里有身体锚点**：自己的手 / 脚 / 膝盖出现在画面下沿
3. **3 个感官位置细节**：脚下看到什么 + 身边看到什么 + 远处看到什么
4. **节拍化动作描述**：先走两步、再侧身、最后抬手——分明动作序列
5. **命名光源和色调锚点**："暖黄路灯偏右、画面整体偏冷蓝"
6. **结尾负面约束**：avoid 漂浮的第三人称视角 / no third-person camera

少一个，模型就会漂回普通跟拍。

## 视点视角（观众用什么身份在看这场戏）

这是分镜师的**上层心智**——决定运镜之前，先决定观众的位置。

| 视角 | 怎么放 | 心理含义 | 怎么写进 prompt |
| --- | --- | --- | --- |
| **跟随视角** | 镜头紧跟主角运动（背影跟拍 / 旁边平移） | 陪伴、参与、共情 | "镜头从人物身后跟随、与主体保持固定距离" |
| **窥探视角** | 长焦 + 框架构图（透过门缝、窗户、柱子缝） | 偷看、距离感、悬疑、不安 | "透过半开的门缝拍摄、长焦压缩空间" |
| **POV / 主观视角** | 镜头模拟主体的眼睛 | 沉浸、代入、生理感受 | "第一人称视角、模拟人物眼睛所见，轻微呼吸感起伏" |
| **打破第四堵墙** | 演员直视镜头 | 伪纪录片、强制参与、戏剧张力 | "人物正面面对镜头、视线直视镜头说话" |
| **上帝视角** | 鸟瞰、俯拍、远景 | 旁观、命运感、战术展开 | "高空俯拍、远景大全景" |

**用户做仿真人剧/广告时**：默认走"跟随视角"最稳；想做悬疑就切"窥探"；做产品口播或品牌片可以试"打破第四堵墙"。

## 风格化运动（节奏 / 帧率 / 抽帧）

AI 视频可以直接响应这些风格化描述，不需要物理摄影机也能做出。

| 风格 | 写法 | 视觉效果 | 工具偏好 |
| --- | --- | --- | --- |
| **升格慢动作** | "整段以慢动作呈现，约 0.5 倍速" / "慢动作特写，火花迸发瞬间被拉长" | 时间被拉长、动作细节放大、情绪扩张 | Seedance 2.0 响应较好 |
| **王家卫抽帧风格** | "画面带强烈拖影感与卡顿抽离感、动态模糊明显" | 既模糊（拖影）又卡顿（跳跃），迷幻、迷离、时间错乱 | Seedance 2.0 较好；可灵会偏向单纯卡顿 |
| **跳切风格化** | "画面带轻微跳跃感、动作不完全连贯，外化人物焦虑" | 焦虑、纠结、慌乱（参考《喜剧之王》翻箱倒柜找钱） | 单镜内不易做到，建议拆段后期接 |
| **碎剪/快慢节奏** | "前段加速、定格瞬间慢放" | 动作戏张力（拳头击中那刻定格） | 单镜难以模拟，更适合拆段做 |
| **自然动态模糊** | "画面带自然动态模糊、不像高速摄影那样定格" | 流畅、人眼真实感、电影感 | 默认就是这种，写法用于强调 |

**关键区分**：
- 写"慢动作"——AI 会拉长时间
- 写"动态模糊"——AI 会让运动物体带拖影
- 写"卡顿/抽帧"——AI 会让画面不连续
- "王家卫抽帧"= **拖影 + 卡顿同时出现**，不只是其中一种

## 单镜内模拟内外反打（对话戏的镜头设计）

AI 单镜不能切换镜头，但 Seedance 2.0 能在单镜内做**景别切换**，可以用来模拟对话戏的内外反打效果：

| 传统手法 | AI 单镜怎么模拟 |
| --- | --- |
| 外反打（带肩） | 中景 + 一侧带另一人肩膀剪影 |
| 内反打（纯单人特写） | 中景缓推到主体面部特写 |
| 推镜配合心理崩溃 | "镜头从中景缓慢推近至面部大特写、背景由清晰渐变为深色虚化" |
| 越轴制造冲突 | 单镜内难做，建议拆成两段独立生成后剪辑 |

## 主体动作（人在做什么）

按"AI 视频能不能稳定处理"分类（**注意**：以下评级针对单一模型的简单场景。Seedance 2.0 在多动作链方向显著强于早期判断、详见 `model-capabilities.md`）：

| 类型 | 举例 | AI 稳定性 | 适合时长 |
| --- | --- | --- | --- |
| **微动作** | 呼吸、眨眼、头发被风吹、手指微动、衣角轻摆 | ⭐⭐⭐⭐⭐ | 任意时长、让画面"活"起来的安全牌 |
| **表情变化** | 平静 → 微笑、平静 → 震惊 | ⭐⭐⭐⭐ | 2-4 秒、慢过渡更自然 |
| **定格瞬间** | 动作停在某个点（转头完成、抬眼看） | ⭐⭐⭐⭐ | 搭配慢运镜用、制造情绪重量 |
| **连贯动作** | 从 A 到 B 的完整动作（走路、伸手、喝水） | ⭐⭐⭐ | 3-5 秒 |
| **大幅动作** | 跑、跳、转身打斗、抛接物体 | ⭐⭐ | 3-5 秒、可灵稍好 |
| **多动作串联** | 转头+端杯+喝一口+放下+笑 | Seedance ⭐⭐⭐ / 可灵 ⭐ | Seedance 2.0 单镜能扛 5+ 拍、详见 model-capabilities.md |

**修正旧铁律**：之前的"一个镜头一个动作"对 Seedance 2.0 已不成立。当前原则——按模型能力上限设计、不要预防性拆段。详见 SKILL.md "单镜能力上限"段。

## 节奏

| 词 | 人话解释 | 适合的情绪 |
| --- | --- | --- |
| 慢节奏 | 时长 5-8 秒、运镜慢、动作少 | 情绪酝酿、沉思、孤独 |
| 中节奏 | 3-5 秒、有运镜或一个完整动作 | 多数场景的默认 |
| 快节奏 | 3 秒内、运镜快或动作密集 | 张力、冲突、动感 |
| 渐强 | 从慢到快、从静到动 | 情绪递进、冲突累积 |
| 渐弱 | 从快到慢、从动到静 | 情绪收敛、冷静下来 |
| 前静后动 | 前半段微动作，后半段主动作 | 最常见稳定结构，"等待 → 发生" |

## 时长建议（按模型/场景查 model-capabilities.md）

**单一数据源**：所有具体时长上限、模型能力边界、推荐时长查 `references/model-capabilities.md`。本表只保留人话理解参考。

| 时长档 | 直觉理解 | 适合 |
| --- | --- | --- |

## 一致性锚点（必写）

可灵和即梦的视频都会在过程中出现**人物脸部逐渐变形**、**服装细节漂移**、**背景元素飘动**。默认在每条提示词末尾加：

> "保持人物面部特征、服装、发型、构图、光线氛围在整段视频中稳定不变。"

改图类场景（首尾帧）还要加：

> "首尾帧之间过渡自然，不出现突兀切换或元素跳变。"

## 常见陷阱（AI 视频不要做的事）

### ❌ 堆砌空泛运镜词

模型听不懂：
- "电影级运镜"
- "大师镜头"
- "震撼运镜"
- "惊艳的镜头语言"
- "专业级画面"

替代方案（写具体可见的动作）：
- "镜头缓慢向人物面部推近"
- "相机从左向右水平移动，跟随人物行走"
- "固定镜头，人物缓慢转头"

### ❌ 让主体做太多动作

- 差：女孩转头 + 端起咖啡 + 喝一口 + 放下 + 对镜头微笑
- 好：女孩缓慢转头望向窗外，完成即停

### ❌ 要求镜头切换 / 剪辑

视频模型生成的是**单个连续镜头**，做不到：
- 镜头切换（"推进过程中突然切到特写"）
- 闪回 / 回忆片段
- 多镜头剪辑
- 字幕 / 旁白（Veo 3 除外）

如果用户需要这些，应该**拆成多个镜头**分别生成，后期剪辑合成。

### ❌ 忽略一致性锚点

永远默认加。不要问用户要不要加，直接加。

### ❌ 写超过 10 秒的单镜头

扭曲率过高。建议拆分。
