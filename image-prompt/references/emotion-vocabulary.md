# 情绪 → 可见动作语料库（图片版）

本文件供 image-prompt skill 使用。SKILL.md 纪律 9 提出的核心原则：模型不会画"紧张"，只会画"手指摩挲杯沿、视线避开、嘴角抿紧"。本文件是 8 维度的扩展查表。

## 使用纪律

1. **不堆砌**。一个画面最多挑 2-3 个维度落地，不是 8 个全用。维度越多，模型注意力越散。
2. **挑能被视觉抓住的**。"生理反应""眼神""表情""手部动作"在静图里最有效；"语气""心理代入"在静图里效力低（视频更适合）。
3. **同一情绪的不同强度，落到不同维度**。微弱紧张 → 只写"手指轻碰杯沿"；强烈紧张 → "嘴角抿紧+下颌绷起+视线避开"组合。
4. **维度词不进【最终提示词】**。"她生理反应明显"是失败输出。要写出生理反应**是什么**：呼吸加快、瞳孔收缩、太阳穴血管浮起。

## 8 维度速查定义

| 维度 | 是什么 | 静图典型写法 |
| --- | --- | --- |
| **内心活动** | 角色心里在想/感受 | 用动作侧写心理（"她看着杯子的时间比该看的时间长" → 暗示走神） |
| **生理反应** | 角色控制不了的身体信号 | 呼吸、瞳孔、脸红、出汗、颤抖、太阳穴血管 |
| **社交习惯** | 这个具体人物的标志性小动作 | 紧张时摸耳垂、思考时咬笔头、不知所措时摸袖口 |
| **表情** | 脸部肌肉的形状 | 嘴角的角度、眉毛的高度、鼻翼的张合、下颌的紧张 |
| **眼神** | 视线方向、聚焦深度、移动轨迹 | 看向何处、是聚焦还是放空、眨眼频率 |
| **语气**（静图退化） | 嘴唇的开合状态、喉部肌肉 | 静图里只剩"嘴唇微启""喉结上下滑动" |
| **心理代入** | 让观众"感同身受"的代入设计 | 视角设计（POV 暗示主角）、距离感、不拍正脸只拍背影 |
| **情感具象化** | 用画面元素象征情感 | 倒下的咖啡杯=失控、撕碎的照片=决裂、空椅子=离开 |

## 常见情绪 → 动作翻译表

### 1. 紧张

- 生理：呼吸加快胸口微微起伏、太阳穴血管轻微浮起、手心轻微出汗（袖口边缘有湿痕）
- 表情：嘴角向内抿成一条直线、下颌肌肉绷紧、鼻翼轻微张合
- 眼神：视线避开对方、瞳孔轻微收缩、眨眼频率高
- 手部：手指无意识摩挲杯沿/笔杆/衣角、指节发白
- 社交习惯（可选）：用手背抹一下后颈、调整袖口

### 2. 愧疚

- 生理：肩膀微微内收下垂、整个身体重心向后退半步
- 表情：嘴角向下但没完全垮、眉头不是皱而是"压低"
- 眼神：避开对方目光、视线落在地面或对方膝盖、眨眼变慢
- 手部：双手在身前轻握或藏到身后、手指无目的弯曲
- 心理代入：常用四分之三背身、不拍正面脸

### 3. 思念

- 内心活动：动作停顿（端起杯子但没喝、翻开书但视线越过书页）
- 生理：嘴唇微启没有发声、轻微的呼吸停顿
- 眼神：视线越过近景物体望向远处或画外、目光不聚焦
- 表情：嘴角弧度很轻、眉毛松弛（不是皱、是放空）
- 情感具象化：手边一杯未喝完的茶/咖啡冒着热气、桌面上摊开一封信、窗外天色已暗但灯没开

### 4. 克制

- 生理：呼吸刻意放慢深长、下颌咬肌轻微凸起
- 表情：嘴唇压成一条线但没用力到发白、眉头中间略皱
- 眼神：目光定在一个点上不移动、长时间不眨眼
- 手部：双手在桌下或身侧握成松松的拳、不是攥紧
- 社交习惯：刻意挺直背、整理衣领

### 5. 破防（情绪溃堤）

- 生理：眼眶发红/泛泪光但泪未流、鼻翼通红、嘴角颤抖
- 表情：嘴角向下拉同时上抬（典型哭脸前奏）、下唇微微外翻
- 眼神：眼神放空一两秒后才聚焦、视线开始游移
- 手部：手捂住嘴/下半张脸/眼睛、手指撑住额头
- 情感具象化：手机刚放下还亮着屏、墙上一张刚被注意到的照片

### 6. 暧昧

- 内心活动：动作和目光不同步（嘴说着别的、眼睛在看对方）
- 生理：脸颊潮红（侧脸更明显）、嘴唇微启
- 眼神：避开正面对视但偷瞥、视线在对方脸上短暂停留然后移开
- 表情：嘴角不自觉上扬但很轻、眉梢微挑
- 距离：两人之间留一拳左右的距离（既不远也不近）、肩膀微微相对

### 7. 失落

- 生理：肩膀下沉、整个身体姿态松垮（不是站姿不是坐姿、是"撑不起来"）
- 表情：嘴角自然下垂（不是用力）、眉毛松弛
- 眼神：视线没有目标地落在中景、对周围环境无反应
- 手部：手垂在身侧或松松放在膝上、手指自然弯曲不发力
- 情感具象化：手中物品（手机/咖啡/票根）停在半空中、画面右侧大量留白

### 8. 疲惫

- 生理：眼下有阴影、嘴唇干涩起皮、脖颈肌肉松弛
- 表情：眉毛之间松弛、嘴角下垂但不悲伤
- 眼神：眼睛半阖、目光不聚焦
- 姿态：靠在椅背/墙上、头微微低着、肩膀往一侧倾斜
- 情感具象化：桌上空咖啡杯排成一列、电脑屏幕仍亮着、外面天已经亮了

### 9. 喜悦（克制版，不是大笑）

- 生理：眼角细纹出现（杜乡式微笑的关键）、脸颊轻微上提
- 表情：嘴角向上但不张嘴、眉梢向外扩
- 眼神：眼神发亮、视线聚焦在喜悦源（人/物）
- 手部：手部动作变轻、可能下意识握住胸前的什么
- 心理代入：常用近景中景、给观众"和角色共享这一刻"的距离

### 10. 愤怒（隐忍版）

- 生理：太阳穴血管浮起、下颌咬肌凸起、呼吸变得深而慢
- 表情：嘴唇紧闭成一线、眉头之间出现垂直皱纹（不是横皱）
- 眼神：瞳孔锁定一个点、眨眼变得极少、目光"硬"
- 手部：手指逐渐握紧、关节发白、手腕静脉浮起
- 情感具象化：身边的玻璃杯/笔被无意识握紧

### 11. 孤独

- 姿态：人物在画面边缘 1/3 处、大片空白指向画外
- 生理：呼吸轻、动作慢、身体收得很小
- 眼神：视线没有具体目标、看向窗外/远处/画外
- 情感具象化：桌上两副餐具但只坐了一个人、双人座只坐一边、电话铃声响着没接
- 心理代入：背影或四分之三背身、不让观众看到正脸

### 12. 决断（下定决心瞬间）

- 生理：深呼吸一次、肩膀从松弛变挺直、下颌微抬
- 表情：嘴唇从抿到松开、眉头从皱到展
- 眼神：视线从游移变聚焦、目光从向下变平视
- 手部：手从无目的状态收到具体动作上（按键、拿钥匙、起身扶椅）
- 情感具象化：手中的纸条/物件被合上/装进口袋

## 非人物场景的 3 维度（空镜 / 静物 / 场景图）

8 维度对没有人的画面失效。改用以下三维度。

### 一、空间留白

主体所占面积 vs 空白面积的对比，决定情绪强度。

| 用法 | 比例 | 情绪 |
| --- | --- | --- |
| 主体被稀释 | 主体 1/4 + 空白 3/4 | 渺小、孤独、被遗忘 |
| 主体被挤压 | 主体顶满画面、四周无呼吸 | 压迫、窒息、紧张 |
| 主体偏一侧 | 主体在 1/3 处、视线/动势指向空白方向 | 怀念、向往、有去处 |
| 主体偏一侧 | 主体在 1/3 处、视线/动势指向画框（撞墙） | 抑郁、闭塞、无路可走 |
| 对称居中 | 主体居中、上下左右等量空间 | 仪式感、命运、肃穆 |
| 三层空间 | 前景 + 中景 + 远景都有内容 | 信息密度、纪实、深入 |

### 二、光影方向

光是有时间和情绪的"语言"。

| 光的特征 | 时间暗示 | 情绪指向 |
| --- | --- | --- |
| 冷白荧光、整体平铺 | 凌晨/便利店/办公室加班 | 疏离、机械、不安 |
| 暖黄从一侧低角度入 | 清晨日出后 1 小时 | 希望、温度、苏醒 |
| 橘红从一侧水平入 | 黄昏日落前 1 小时 | 温暖、留恋、即将结束 |
| 蓝紫弥漫无明确方向 | 日落后 30 分钟（蓝色时刻） | 安静、思念、过渡 |
| 暖黄孤点 + 周围深暗 | 夜里 | 孤立、避风港、私密 |
| 月光冷蓝从窗入 + 室内全暗 | 深夜 | 失眠、隐秘、不安 |
| 霓虹红蓝交错 + 雨水反光 | 夜晚都市 | 迷离、欲望、王家卫式 |
| 顶光硬光 + 大面积阴影 | 中午/审讯室 | 戏剧、审视、不留情 |

### 三、物体的叙事暗示

物品的状态隐藏故事。

| 物体状态 | 暗示 |
| --- | --- |
| 还冒热气的咖啡 + 没人 | 刚有人离开 |
| 倒地的椅子 / 翻倒的杯子 | 冲突发生过 |
| 两副餐具但只剩一人 | 关系缺席、等待 |
| 摊开的信 + 半空的笔 | 未完成的告白 / 决心 |
| 票根 / 登机牌摆在桌上 | 即将离开 / 已经回来 |
| 雨伞滴水放在门口 + 空鞋架 | 刚到家 |
| 床的一半被睡过 + 另一半整齐 | 独自起床 |
| 戒指放在桌上 + 不在手上 | 关系状态变化 |
| 烟灰缸里很多烟蒂 | 等了很久 / 焦虑了很久 |
| 桌上一杯水半杯 + 杯口有口红印 | 之前有过对话 |

## 反例集

### ❌ 维度堆叠

错：她内心紧张、生理反应明显、有摸耳垂的社交习惯、表情紧绷、眼神躲闪、嘴唇抿紧、心理代入感强、用空荡的桌子做情感具象化

为什么错：8 维度全用了，没有焦点。模型不知道画什么最重。

正：她嘴角抿紧、视线避开对方落在杯沿、手指无意识地摩挲杯壁。

挑了 3 个最强信号（表情+眼神+手），其他维度让画面自己呼吸。

### ❌ 用情绪词替代动作

错：她崩溃地坐在那里。

正：她坐在椅子上，肩膀塌下来，手机还亮着屏幕扣在桌面上，眼眶发红但泪没流出来，鼻翼通红。

### ❌ 矛盾信号叠加

错：她紧张地放声大笑，目光坚定地避开对方。

为什么错："大笑"和"紧张"、"坚定"和"避开"是矛盾的，模型无所适从。

正：要么"她笑得勉强、嘴角的弧度比眼睛先到位"（紧张的笑）；要么"她爽朗大笑，肩膀放松，眼角的皱纹深陷"（放松的笑）。先定情绪基调再挑动作。
