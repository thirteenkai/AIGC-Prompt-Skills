# 故事板审核与视频提示词模板

## 故事板审核

用户上传故事板后，先审核，不要直接生成视频提示词。

```text
【故事板审核】

结论：通过 / 需要重抽 / 可进入视频但需修正提示词

1. 分镜顺序：
2. 叙事完整性：
3. 关键动作：
4. 情绪递进：
5. 角色/道具一致性：
6. 中文说明可读性：
7. 易误解点：
8. 建议：
```

通过标准：

- Cut 顺序和剧情一致
- 关键动作出现
- 情绪变化能看懂
- 没有把角色关系误解
- 中文说明清楚
- 局部小穿帮不影响叙事

## 资产前置规则

视频提示词必须先写清资产表：

```text
【参考资产】
@图1 = {资产名称，如街边摊位环境}
@图2 = {资产名称，如主角}
@图3 = {资产名称，如关键道具}
@音频1 = {音频名称，如角色音色或节拍参考}
```

后文直接用资产名称，不要每次重复 @。

## A/B 分段视频规则

如果故事板被判断为超过 15 秒承载范围，视频提示词必须输出两段：

- 视频 A：完成前半段因果，并在结尾留下可衔接的动作、视线、道具或角色状态。
- 视频 B：开头承接视频 A 的最后状态，不重新开局，不改变角色位置和道具状态。
- 两段共用同一份【参考资产】。第二段可以写“沿用视频 A 参考资产”，不要重复大量资产说明。
- 不要把 A/B 写成两个互相独立的故事；它们是一段剧情的上下半场。

## 固定结构

```text
参考上传的故事板和故事板描述，生成一段 {10/15} 秒以内的{账号风格}视频。

【参考资产】
@图1 = ____
@图2 = ____
@图3 = ____

【参考依据】
故事板用于确定镜头顺序、角色关系、场景空间、情绪变化和关键动作。
视频必须严格按故事板 Cut 顺序推进：{Cut 顺序摘要}。
如果故事板画面细节和文字描述冲突，以文字描述为准。

【叙事核心】
{这段视频必须让观众看懂什么}

【情绪曲线】
{情绪 A -> 情绪 B -> 情绪 C}

【视频分镜】

镜头1：{标题}
{镜头与机位}
{画面内容}
{主体动作，写清开始状态、动作过程、动作结果}
{情绪表现}
{必须保留/不要误解}

镜头2：{标题}
...

【声音设计】
{按镜头写环境音、动作音、台词、后期字幕、收款音效等}

【画面风格】
{账号风格、光线、质感、环境要求}

【关键约束】
{列出 8-12 条禁止跑偏项}
```

拆分时使用：

```text
【视频 A 提示词】
目标：{A 段要让观众看懂什么}
结尾衔接状态：{B 段必须承接的角色位置、道具状态、情绪状态}
{按固定结构输出 A 段视频分镜}

【视频 B 提示词】
开头承接状态：{承接 A 段最后一镜}
目标：{B 段要解决什么问题}
{按固定结构输出 B 段视频分镜}
```
